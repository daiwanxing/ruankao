# 第八章 监理概论

## 8.1 监理基本概念

### 一、教学目标

掌握信息系统工程监理的基本概念、工作依据、主要内容和监理工程师的职业道德要求。

### 二、核心讲解

#### 1. 监理的基本概念

**信息系统工程监理**是指依法设立且具有相应资质的信息系统工程监理单位，受业主单位委托，依据国家有关法律法规、技术标准和监理合同，对信息系统工程项目实施的监督管理。

**监理的核心目标**：
- 协助业主单位确保工程项目在**合理的时间、合理的费用**下完成
- 保证工程项目的**质量**符合国家或行业相关标准
- 实现项目的**投资效益最大化**

#### 2. 监理的工作依据

| 依据类型 | 具体内容 |
|----------|----------|
| **法律法规** | 《建筑法》、《合同法》、《招标投标法》、《政府采购法》等 |
| **技术标准** | 信息系统工程相关国家标准、行业标准 |
| **监理合同** | 业主单位与监理单位签订的监理服务合同 |
| **项目文件** | 可行性研究报告、设计文件、招标文件等 |
| **三方合同** | 业主单位与承建单位签订的工程合同 |

#### 3. 监理的主要内容

**监理工作"三控两管一协调"**：

| 核心内容 | 含义 | 记忆要点 |
|----------|------|----------|
| **质量控制** | 监督工程质量符合标准 | 📐 **质量** |
| **进度控制** | 监督工程进度按计划进行 | 📅 **进度** |
| **投资控制** | 监督投资合理使用 | 💰 **投资** |
| **合同管理** | 管理工程相关合同 | 📋 **合同** |
| **信息管理** | 管理项目信息资料 | 📊 **信息** |
| **组织协调** | 协调各方关系 | 🤝 **协调** |

#### 4. 监理的范围

**必须实施监理的项目**：
1. 国家重点建设项目
2. 关系公共利益的基础设施项目
3. 使用国有资金投资项目
4. 使用国际组织贷款、援助资金的项目

**监理等级划分**：

| 等级 | 投资规模 | 监理范围 |
|------|----------|----------|
| **甲级** | 3000万元以上 | 可在全国范围内承接监理项目 |
| **乙级** | 500万-3000万元 | 可在省级范围内承接监理项目 |
| **丙级** | 500万元以下 | 可在省级范围内承接监理项目 |

### 三、通俗例子

**监理例子**：
就像房屋装修时请的第三方监理，业主自己不专业，怕被装修公司偷工减料或拖延工期，所以请专业监理来监督施工方，确保装修质量、进度和费用都符合要求。

**三控两管一协调例子**：
就像管理一个建筑工地：
- 三控：控制质量（建材合格）、控制进度（按时完工）、控制成本（不超预算）
- 两管：管合同（按合同办事）、管资料（记录存档）
- 一协调：协调业主、施工方、设计方等各方关系

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **监理目标** | 合理时间、合理费用、符合质量 | 时效质量成本 |
| **监理依据** | 法律法规、技术标准、监理合同、项目文件、三方合同 | 五依据 |
| **三控两管一协调** | 质量、进度、投资、合同、信息、组织协调 | 三控两管一协调 |
| **监理等级** | 甲级（全国）、乙级（省级）、丙级（省级） | 等级划分 |
| **监理范围** | 国家重点项目、国有资金项目、公共利益项目 | 重点项目 |

## 8.2 监理组织与人员

### 一、教学目标

掌握监理组织的设置原则、监理人员的职责分工以及监理工程师的职业道德要求。

### 二、核心讲解

#### 1. 监理组织设置原则

| 原则 | 含义 | 说明 |
|------|------|------|
| **专业分工** | 根据专业领域设置监理岗位 | 硬件、软件、网络、安全等 |
| **层次适当** | 监理组织层级合理 | 总监理工程师→总监代表→专业监理工程师→监理员 |
| **统一指挥** | 单一领导，避免多头指挥 | 确保指令统一 |
| **跨度合理** | 每个管理者管理人数适当 | 一般6-8人为宜 |
| **精干高效** | 人员精简，提高效率 | 避免人浮于事 |

#### 2. 监理人员职责分工

**总监理工程师职责**：

| 职责 | 具体内容 |
|------|----------|
| 全面负责 | 负责监理项目的总体管理和协调 |
| 人员调配 | 调配监理人员和工作资源 |
| 审核批准 | 审批监理规划、实施细则 |
| 协调沟通 | 协调业主、承建单位关系 |
| 签发指令 | 签发开工令、停工令、复工令 |
| 争议处理 | 处理项目中的重大争议 |

**专业监理工程师职责**：

| 职责 | 具体内容 |
|------|----------|
| 专业负责 | 负责本专业领域的监理工作 |
| 技术指导 | 指导监理员开展具体工作 |
| 资料管理 | 编写本专业的监理资料 |
| 质量检查 | 检查本专业的工程质量 |
| 问题处理 | 处理本专业的一般问题 |

**监理员职责**：

| 职责 | 具体内容 |
|------|----------|
| 现场监理 | 在施工现场进行日常监理 |
| 记录工作 | 记录监理日志、旁站记录 |
| 问题报告 | 发现问题及时向上报告 |
| 资料收集 | 收集整理监理资料 |

#### 3. 监理人员素质要求

**监理工程师应具备的素质**：

| 素质类型 | 具体要求 |
|----------|----------|
| **专业素质** | 扎实的信息系统专业知识 |
| **管理素质** | 具备项目管理能力 |
| **协调素质** | 良好的沟通协调能力 |
| **职业操守** | 公正、廉洁、诚实守信 |

#### 4. 监理工程师职业道德

**监理工程师职业道德规范**：

| 道德要求 | 具体内容 |
|----------|----------|
| **守法遵规** | 遵守国家法律法规和行业规范 |
| **公正独立** | 独立、公正地开展监理工作 |
| **廉洁从业** | 廉洁自律，不谋取私利 |
| **诚实守信** | 诚实守信，维护行业信誉 |
| **保密义务** | 对项目信息保密 |
| **回避制度** | 与被监理单位有利害关系时回避 |

### 三、通俗例子

**监理组织例子**：
就像一个医院的组织架构：
- 院长（总监理工程师）→ 全面管理
- 各科室主任（专业监理工程师）→ 专业负责
- 主治医生、护士（监理员）→ 具体执行

**职业道德例子**：
就像医生不能收红包一样，监理工程师也不能收受承建单位的贿赂，必须公正独立地开展监理工作。

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **组织设置原则** | 专业分工、层次适当、统一指挥、跨度合理、精干高效 | 五原则 |
| **监理人员层级** | 总监理工程师→总监代表→专业监理工程师→监理员 | 四层级 |
| **总监理工程师职责** | 全面管理、人员调配、审核批准、协调沟通、签发指令 | 六职责 |
| **职业道德要求** | 守法遵规、公正独立、廉洁从业、诚实守信、保密回避 | 五要求 |

## 8.3 监理工作内容与方法

### 一、教学目标

掌握监理工作的具体内容、主要方法和工作流程。

### 二、核心讲解

#### 1. 监理工作阶段划分

**监理工作贯穿项目全生命周期**：

| 项目阶段 | 监理工作内容 |
|----------|--------------|
| **项目决策阶段** | 可行性研究评审、项目建议书审核 |
| **设计阶段** | 设计方案评审、设计进度和质量管理 |
| **招标阶段** | 招标文件审查、招标过程监督 |
| **施工阶段** | 质量控制、进度控制、投资控制 |
| **验收阶段** | 验收测试组织、验收文档审核 |
| **运维阶段** | 运维服务质量监督 |

#### 2. 监理工作方法

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| **审核** | 审核各类文档、方案、报告 | 设计方案、进度计划等 |
| **检查** | 现场检查、巡视检查 | 工程质量、安全措施 |
| **试验** | 各类测试、检测 | 软件测试、设备检测 |
| **旁站** | 现场全程监督关键工序 | 隐蔽工程、关键节点 |
| **签认** | 对重要事项签字确认 | 验收签字、付款签认 |
| **协调** | 协调各方关系和争议 | 解决纠纷、统一意见 |

#### 3. 监理主要工作

**(1) 质量控制**

| 控制阶段 | 控制内容 |
|----------|----------|
| **事前控制** | 审核承建单位资质、检查质量保证体系 |
| **事中控制** | 巡视检查、旁站监理、平行检验 |
| **事后控制** | 验收测试、质量评估 |

**(2) 进度控制**

| 控制类型 | 具体措施 |
|----------|----------|
| **进度计划审核** | 审核项目进度计划的合理性和可行性 |
| **进度跟踪** | 定期检查实际进度与计划进度的偏差 |
| **进度调整** | 分析偏差原因，督促采取纠偏措施 |
| **进度报告** | 编制进度监理报告，向业主汇报 |

**(3) 投资控制**

| 控制环节 | 具体措施 |
|----------|----------|
| **投资审核** | 审核工程量清单、工程变更 |
| **投资监控** | 跟踪投资使用情况 |
| **投资支付** | 审核工程款支付申请 |
| **投资结算** | 审核工程结算报告 |

#### 4. 监理工作流程

```
监理工作流程：
项目承接 → 组建监理部 → 编制监理规划 → 实施监理工作 → 监理资料整理 → 项目总结
    ↓           ↓              ↓              ↓              ↓             ↓
  签订合同   人员配置      制定计划     日常监理      归档资料      经验总结
```

### 三、通俗例子

**监理方法例子**：
就像检查学生学习情况：
- 审核（看作业）→ 检查（课堂巡视）→ 试验（考试）→ 旁站（监考）→ 签认（批改签字）

**质量控制例子**：
就像做饭：
- 事前控制：检查食材是否新鲜
- 事中控制：监督烹饪过程
- 事后控制：品尝检查是否熟透

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **监理方法** | 审核、检查、试验、旁站、签认、协调 | 六方法 |
| **质量控制三阶段** | 事前控制、事中控制、事后控制 | 三阶段 |
| **进度控制措施** | 计划审核、跟踪、调整、报告 | 四措施 |
| **监理工作流程** | 项目承接→组建→编制→实施→整理→总结 | 六步骤 |

### 二、易混淆概念辨析

#### 1. 质量控制 vs 进度控制 vs 投资控制

| 概念 | 控制目标 | 控制重点 |
|------|----------|----------|
| **质量控制** | 确保工程质量符合标准 | 原材料、施工工艺、验收测试 |
| **进度控制** | 确保工程按计划时间完成 | 关键路径、资源调配、进度偏差 |
| **投资控制** | 确保投资合理使用 | 工程量变更、付款审核、结算审计 |

#### 2. 事前控制 vs 事中控制 vs 事后控制

| 概念 | 时机 | 内容 | 目的 |
|------|------|------|------|
| **事前控制** | 事件发生前 | 审核资质、检查准备 | 预防问题发生 |

### 四、真题演练

#### 题型一：概念理解题

**例题**：监理工作的核心目标是（ ）
A. 降低工程成本
B. 确保在合理的时间和费用下完成符合质量标准的工程
C. 加快工程进度
D. 减少监理人员工作量

**解析**：监理的核心目标是帮助业主在合理的时间、合理的费用下完成符合质量标准的工程。答案选B。

#### 题型二：分类记忆题

**例题**：监理工作的"三控"包括（ ）
A. 质量控制、进度控制、人员控制
B. 质量控制、进度控制、投资控制
C. 质量控制、成本控制、安全控制
D. 进度控制、投资控制、合同控制

**解析**："三控"指质量控制、进度控制、投资控制。答案选B。

#### 题型三：原则应用题

**例题**：监理组织设置中，"统一指挥"原则是指（ ）
A. 统一监理标准
B. 单一领导，避免多头指挥
C. 统一监理费用
D. 统一工作方法

**解析**："统一指挥"是指在监理组织中，每个下级只接受一个上级的指令，避免多头指挥造成混乱。答案选B。
