# 第一章 信息化发展

## 1.1 信息与信息化

### 一、教学目标

掌握信息的基本概念、信息化的内涵与特征，以及国家信息化战略规划，为后续学习打下坚实基础。

### 二、核心讲解

#### 1. 信息的概念

**数据与信息的关系**：
- **数据**：由原始事实组成，是信息的载体
- **DIKW模型**：数据(Data) → 信息(Information) → 知识(Knowledge) → 智慧(Wisdom)

**有价值的信息特征（12个特征）**：
便捷性、准确性、完整性、经济性、灵活性、相关性、可靠性、安全性、简单性、及时性、可检验性

#### 2. 信息化的基本内涵

信息化的**口诀记忆**："主权、常识、空域、手工、涂改、飙升"

| 维度 | 内涵 |
|------|------|
| 主体 | 全体社会成员（政府、企业、事业、团体和个人） |
| 时域 | 是一个长期的过程 |
| 空域 | 政治、经济、文化、军事和社会的一切领域 |
| 手段 | 基于现代信息技术的先进社会生产工具 |
| 途径 | 创建信息时代的社会生产力 |
| 目标 | 使国家的综合实力、社会的文明素质和人民的生活质量全面提升 |

#### 3. 国家信息化战略与规划

**组织信息化趋势**：产品信息化、产业信息化、社会生活信息化、国民经济信息化

**三步走战略**：
- **第一步（到2020年）**：核心关键技术部分领域达到国际先进水平
- **第二步（到2025年）**：建成国际领先的移动通信网络
- **第三步（到21世纪中叶）**：信息化全面支撑社会主义现代化建设

### 三、通俗例子

**DIKW模型的例子**：
- **数据**：你今天走了10,000步（原始数字）
- **信息**：今天比昨天多走了2,000步（有了比较和意义）
- **知识**：连续一周运动步数增加，说明身体状况在改善（规律认知）
- **智慧**：知道如何通过合理运动来保持健康（运用知识的能力）

**信息化主体例子**：
就像一个城市的发展不仅需要政府的规划，还需要企业、学校、医院、社区等各方参与，信息化的主体也是全社会成员。

### 四、知识点速记

1. **DIKW模型**：数据→信息→知识→智慧
2. **信息化六要素**：主体（全社会）、时域（长期）、空域（全领域）、手段（信息技术）、途径（社会生产力）、目标（综合实力提升）
3. **信息化三步走**：2020年（部分先进）、2025年（国际领先）、21世纪中叶（全面支撑）
4. **信息化发展趋势**：产品→产业→社会生活→国民经济

## 1.2 信息基础设施

### 一、教学目标

掌握新型基础设施建设（大基建）七大领域、云计算服务类型、大数据4V特征、物联网特点、工业互联网四大体系、区块链核心技术、人工智能分类以及VR/AR区别。

### 二、核心讲解

#### 1. 新型基础设施建设（新基建）

**定义**：以新发展理念为引领，以技术创新为驱动，以信息网络为基础，面向高质量发展，提供数字转型、智能升级、融合创新服务的基础设施体系。

**新基建七大领域**：5G基建、特高压、城际高速铁路和城际轨道交通、新能源汽车充电桩、大数据中心、人工智能、工业互联网

**三大分类**：

| 类型 | 关键词 | 主要内容 |
|------|--------|----------|
| 新型基础设施 | 技术新 | 5G、物联网、工业互联网、卫星互联网、人工智能、云计算、区块链、数据中心、智能计算中心 |
| 融合基础设施 | 应用新 | 智能交通基础设施、智慧能源基础设施 |
| 创新基础设施 | 平台新 | 重大科技基础设施、科教基础设施、产业技术创新基础设施 |

#### 2. 云计算

**特征**：超大规模、高可扩展性、虚拟化、高可靠性、通用性、廉价性、灵活定制

**服务类型分类**：

| 层次 | 全称 | 说明 |
|------|------|------|
| IaaS | 基础设施即服务 | 以服务形式提供服务器、存储和网络硬件 |
| PaaS | 平台即服务 | 包含数据库、中间件及开发工具，以服务形式通过互联网提供 |
| SaaS | 软件即服务 | 将应用程序以服务形式提供给用户，通过浏览器提供给成千上万的用户 |

**部署范围**：公有云、私有云、混合云

#### 3. 大数据

**4V特征**：
- **Volume（规模性）**：数据量大
- **Variety（多样性）**：数据类型多
- **Value（价值）**：价值密度低
- **Velocity（速度）**：处理速度快

**结构分类**：
- **结构化数据**：符合关系型数据库，二维表
- **半结构化数据**：不符合关系型数据库
- **非结构化数据**：没有固定结构（如文本、图片、各类报表）

**Hadoop生态系统**：
- **HDFS**：分布式文件存储系统
- **MapReduce**：计算模型，用于大规模数据集并行运算
- **Yarn**：资源管理

#### 4. 物联网（IoT）

**特点**：互连、识别与通信、智能化

**产业六个特点**：感知识别普适化、异构设备互连化、互联网终端规模化、管理调控智能化、应用服务链条化、经济发展跨越化

**分类**：
- **按部署方式**：私有物联网、公有物联网、社区物联网、混合物联网
- **按传输速率**：高速率、中速率、低速率业务

#### 5. 工业互联网

**四大体系**：
- **网络体系（基础）**：网络互连、数据互通、标识解析
- **平台体系（中枢）**：边缘层、IaaS层、PaaS层、SaaS层
- **数据体系（要素）**：价值创造的源泉
- **安全体系（保障）**：设备、控制、网络、平台、工业应用的整体保护

#### 6. 区块链

**特点**：多方协作、不可篡改、可追溯

**分类**：公有链、私有链、联盟链

**核心技术**：

| 技术 | 说明 |
|------|------|
| 分布式存储 | 分布式公共账本，点对点记账系统 |
| 共识机制 | PoW、PoS、DPoS、Paxos、PBFT等 |
| 智能合约 | 基于预定义事件触发、不可篡改、自动执行的计算机协议 |
| 加密算法 | 对称加密（如DES）和非对称加密（如RSA） |
| 跨链技术 | 将区块链数据安全转移到另一条区块链 |
| 分片技术 | 将数据库分割为不同片区以达到系统扩容 |

#### 7. 人工智能

**分类**：
- **弱人工智能**：不能真正实现推理和解决问题，无自主意识
- **强人工智能**：真正能思维的智能机器，有自我意识

**特征**：由人类设计，为人类服务；能感知环境；有适应特性

#### 8. 虚拟现实（VR）和增强现实（AR）

**区别**：

| 区别 | VR | AR |
|------|----|----|
| 设备 | 位置追踪器、数据手套、动作捕捉系统、数据头盔 | 3D摄像头，只要安装AR软件，带摄像头的产品都可以进行AR体验 |
| 技术 | 绘图相关技术，关注沉浸感 | 计算机视觉技术，对真实场景进行3D建模再处理，重视CPU处理能力 |
| 应用场景 | 游戏、娱乐、教育、社交 | 生活、工作、生产等场景 |

### 三、通俗例子

**云计算例子**：
- **IaaS**：像租房子，提供基本的居住空间
- **PaaS**：像租房+装修，提供装修好的房子
- **SaaS**：像住酒店，直接入住享受服务

**大数据例子**：
就像淘宝分析你的购物行为，虽然单看一次购买行为数据量很小，但通过分析海量用户的购买数据，就能发现消费趋势，这就是大数据"4V"的体现。

**物联网例子**：
智能家居就是物联网的典型应用，冰箱、空调、灯光等设备互相连接，可以通过手机远程控制。

**区块链例子**：
就像村里记账，以前只有村长一个人记账（中心化），现在每家每户都有一本一模一样的账本（分布式），如果有人想篡改，需要同时修改大部分账本，这几乎不可能。

### 四、知识点速记

1. **新基建七大领域**：5G、特高压、城际轨道、充电桩、大数据、AI、工业互联网
2. **新基建三大类**：技术新（信息基础设施）、应用新（融合基础设施）、平台新（创新基础设施）
3. **云计算三层**：IaaS（基础设施）、PaaS（平台）、SaaS（软件）

## 1.3 信息化应用

### 一、教学目标

掌握数字政府、数字经济、智慧城市和数字乡村四个重点应用领域的主要特征、发展历程和应用场景。

### 二、核心讲解

#### 1. 数字政府

**四大特征**：
- 信息传播的平等化
- 社会生活的全面数据化
- 政府服务的精准化
- 政府治理的智慧化

**发展历程**：
第一阶段：电子政府 → 第二阶段：网络政府 → 第三阶段：数字政府

**三大应用场景**：
- **政务服务一网通办**：群众办事像网购一样方便
- **城市治理一网统管**：城市运行"一网统管"
- **政府运行一网协同**：政府内部协同办公

#### 2. 数字经济

**"四化"框架**：
- **数字产业化**（先导产业）：如电子信息制造、电信业等
- **产业数字化**（主阵地）：传统产业数字化转型
- **数字化治理**（治理模式）：政府治理数字化
- **数据价值化**（关键生产要素）：数据成为重要生产要素

**三大定律**：
- **梅特卡夫定律**：网络价值等于其节点数的平方
- **摩尔定律**：计算机芯片的处理能力每18个月就翻一番
- **达维多定律**：进入市场的第一代产品能够自动获得50%市场份额

**五个基本特征**：数字化、网络化、智能化、商业化、共享化

#### 3. 智慧城市

**发展历程**：数字城市 → 智慧城市 → 新型智慧城市 → 数字孪生城市

**新型智慧城市特征**：
- 开放
- 共建
- 共享
- 服务均等化
- 城市特色化

**应用系统**：
- 智慧政府
- 智慧民生
- 智慧交通
- 智慧产业
- 智慧经济

**信息基础设施**：
- 城市骨干网
- 无线城市
- 三网融合（电信网、广播电视网和互联网）

#### 4. 数字乡村

**定义**：在农村普及数字化、信息化的各种发展模式

**六大方面**：
- 乡村信息基础设施
- 乡村要素数据信息互通
- 乡村产业数字化
- 乡村治理数字化
- 乡村民生数字化
- 生态宜居数字化

### 三、通俗例子

**数字政府例子**：
就像"浙里办"APP，老百姓办事不用跑腿，在手机上就能办理各种证件，这就是"一网通办"的典型应用。

**数字经济例子**：
直播带货就是数字经济的典型代表，通过网络直播销售商品，实现了数字化、网络化、商业化的融合。

**智慧城市例子**：
杭州的城市大脑，通过AI分析交通流量，实时调整红绿灯时间，有效缓解交通拥堵，这就是智慧城市中"城市治理一网统管"的体现。

**数字乡村例子**：
农民通过手机APP学习种植技术、查看农产品价格、销售农产品，这就是数字乡村中"乡村民生数字化"的应用。

### 四、知识点速记

1. **数字政府三阶段**：电子政府→网络政府→数字政府
2. **数字政府三场景**：一网通办、一网统管、一网协同
3. **数字经济四化**：数字产业化、产业数字化、数字化治理、数据价值化
4. **数字经济三定律**：梅特卡夫（网络价值）、摩尔（芯片性能）、达维多（首代产品）
5. **智慧城市四阶段**：数字城市→智慧城市→新型智慧城市→数字孪生城市
6. **数字乡村六方面**：信息基础设施、数据互通、产业数字化、治理数字化、民生数字化、生态数字化
