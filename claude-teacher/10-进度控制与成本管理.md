# 第十章 进度控制与成本管理

## 10.1 进度控制概述

### 一、教学目标

掌握信息系统工程进度控制的基本概念、进度计划编制方法和进度控制措施。

### 二、核心讲解

#### 1. 进度控制基本概念

**进度定义**：项目在时间维度上的安排和执行情况。

**进度控制目标**：
- 确保项目在**合同约定的时间**内完成
- 保证项目各阶段、各里程碑按计划执行
- 及时发现和处理进度偏差

**影响进度的因素**：

| 因素类型 | 具体因素 |
|----------|----------|
| **人的因素** | 人员能力不足、人员流动、沟通不畅 |
| **技术因素** | 技术方案不合理、技术难题未解决 |
| **物资因素** | 设备不到位、材料供应延迟 |
| **环境因素** | 自然环境、不可抗力 |
| **管理因素** | 计划不合理、协调不到位 |
| **业主因素** | 需求变更、付款延迟 |

#### 2. 进度计划编制

**进度计划类型**：

| 计划类型 | 说明 | 适用场景 |
|----------|------|----------|
| **项目总进度计划** | 项目整体进度安排 | 项目启动阶段 |
| **阶段进度计划** | 各阶段的详细进度安排 | 各阶段开始前 |
| **月度进度计划** | 每月的具体工作安排 | 每月初 |
| **周进度计划** | 每周的具体工作安排 | 每周初 |

**进度计划编制方法**：

| 方法 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| **甘特图** | 用条形图表示任务和时间 | 直观易懂 | 难以表达任务依赖 |
| **关键路径法** | 找出关键路径，重点管理 | 明确重点 | 计算复杂 |
| **里程碑计划** | 以里程碑为节点的进度计划 | 目标明确 | 细节不足 |
| **网络计划技术** | 用网络图表示任务关系 | 表达清晰 | 专业性强 |

#### 3. 进度控制措施

**进度控制措施类型**：

| 措施类型 | 具体措施 |
|----------|----------|
| **组织措施** | 建立健全进度控制组织体系、明确职责分工 |
| **技术措施** | 优化技术方案、采用先进工具 |
| **经济措施** | 设立进度奖励、调整付款计划 |
| **合同措施** | 明确进度要求、设立进度违约条款 |
| **信息措施** | 建立进度报告制度、加强进度沟通 |

### 三、通俗例子

**进度控制例子**：
就像组织一次旅游：
- 制定行程计划（进度计划）
- 每天检查是否按时出发（进度检查）
- 发现迟到的人采取措施（进度纠偏）
- 调整后续行程确保按时到达目的地（进度调整）

**甘特图例子**：
就像课程表，用横条表示每节课的时间，一眼就能看出什么时候上什么课。

## 10.2 进度监控与调整

### 一、教学目标

掌握进度监控的方法、进度偏差分析以及进度调整的措施。

### 二、核心讲解

#### 1. 进度监控方法

**进度监控方式**：

| 监控方式 | 说明 | 频率 |
|----------|------|------|
| **日常监控** | 日常工作进度跟踪 | 每日 |
| **定期检查** | 阶段性进度检查 | 每周/每月 |
| **里程碑检查** | 关键节点进度确认 | 按里程碑 |
| **专项检查** | 针对特定问题检查 | 按需 |

**进度监控方法**：

| 方法 | 说明 |
|------|------|
| **进度报告** | 通过报告了解进度情况 |
| **进度会议** | 通过会议讨论进度问题 |
| **现场检查** | 现场核实实际进度 |
| **进度曲线** | 用曲线对比计划与实际 |

#### 2. 进度偏差分析

**进度偏差计算**：

| 偏差类型 | 计算公式 | 含义 |
|----------|----------|------|
| **进度偏差** | 计划工作量 - 已完成工作量 | 偏差的工作量 |
| **进度绩效指数** | 已完成工作量 / 计划工作量 | 大于1表示超前 |
| **关键路径偏差** | 关键路径上任务的偏差 | 影响项目总工期 |

**偏差原因分析**：

| 偏差原因 | 说明 |
|----------|------|
| **计划过于乐观** | 计划本身不合理 |
| **执行力度不足** | 人员工作效率低 |
| **资源不足** | 人力、设备等资源不够 |
| **外部干扰** | 业主变更、供应商延迟 |
| **不可预见因素** | 天气、政策等 |

#### 3. 进度调整措施

**进度调整原则**：

| 原则 | 说明 |
|------|------|
| **关键路径优先** | 优先调整关键路径上的任务 |
| **赶工优先** | 先考虑赶工（增加资源） |
| **变更最小** | 尽量减少变更范围 |
| **风险可控** | 确保调整后风险可控 |

**进度调整措施**：

| 措施 | 说明 | 适用情况 |
|------|------|----------|
| **赶工** | 增加资源投入、加班赶工 | 短期延误 |
| **并行作业** | 将串行任务改为并行 | 资源允许 |
| **快速跟进** | 提前开始后续任务 | 风险可控 |
| **缩减范围** | 适当减少非必要功能 | 业主同意 |
| **重新规划** | 重新制定进度计划 | 严重延误 |

### 三、通俗例子

**进度偏差分析例子**：
就像开车去目的地：
- 原计划1小时到达（计划）
- 半小时后走了20公里，应该走30公里（偏差）
- 分析原因：堵车/开慢了/路线问题
- 采取措施：换道/加速/重新规划路线

**进度调整例子**：
就像赶火车：
- 发现可能要迟到（进度偏差）
- 加快速度（赶工）
- 打车去火车站（并行/快速跟进）
- 改签下一班（重新规划）

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **监控方式** | 日常监控、定期检查、里程碑检查、专项检查 | 四方式 |
| **偏差分析** | 进度偏差、进度绩效指数、关键路径偏差 | 三分析 |
| **调整原则** | 关键路径优先、赶工优先、变更最小、风险可控 | 四原则 |
| **调整措施** | 赶工、并行、快速跟进、缩减范围、重新规划 | 五措施 |

## 10.3 成本管理概述

### 一、教学目标

掌握信息系统工程成本管理的基本概念、成本估算方法和成本控制措施。

### 二、核心讲解

#### 1. 成本管理基本概念

**成本定义**：完成项目所需的各种资源的价值总和。

**成本类型**：

| 成本类型 | 说明 | 举例 |
|----------|------|------|
| **直接成本** | 可直接计入项目的成本 | 人员工资、设备采购 |
| **间接成本** | 需要分摊的项目成本 | 管理费、办公费 |
| **固定成本** | 不随工作量变化的成本 | 房租、设备折旧 |
| **变动成本** | 随工作量变化的成本 | 材料费、差旅费 |
| **沉没成本** | 已发生且无法收回的成本 | 前期的错误投入 |

**成本管理目标**：
- 确保项目在**批准的预算内**完成
- 合理分配和使用项目资源
- 及时发现和控制成本偏差

#### 2. 成本估算

**成本估算方法**：

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| **类比估算** | 参考类似项目的成本进行估算 | 项目早期、信息不足 |
| **参数估算** | 利用统计关系估算 | 有历史数据 |
| **自下而上估算** | 从细节逐层汇总 | 有详细分解 |
| **三点估算** | 基于乐观、悲观、最可能值估算 | 不确定性高 |
| **专家判断** | 由专家凭经验估算 | 无历史数据 |

**成本估算步骤**：

```
识别成本构成 → 估算各项成本 → 汇总成本总额 → 审核估算结果
```

#### 3. 成本控制

**成本控制措施**：

| 措施类型 | 具体措施 |
|----------|----------|
| **组织措施** | 建立健全成本控制体系、明确职责 |
| **技术措施** | 优化方案、采用新技术降低造价 |
| **经济措施** | 编制资金使用计划、严格审核支出 |
| **合同措施** | 明确合同价格条款、控制变更 |

**成本控制流程**：

```
预算编制 → 成本监控 → 偏差分析 → 纠偏措施 → 成本报告
```

### 三、通俗例子

**直接成本 vs 间接成本例子**：
就像开餐厅：
- 直接成本：厨师的工资、食材费用（直接用于菜品）
- 间接成本：经理工资、房租、水电（需要分摊）

**类比估算例子**：
就像估算装修费用，听说朋友家100平米花了20万，自己家120平米，大概估算24万左右。

| **成本控制措施** | 组织、技术、经济、合同 | 四措施 |
| **成本管理目标** | 预算内完成、合理分配、及时控制 | 三目标 |

## 10.4 成本监控与控制

### 一、教学目标

掌握成本监控的方法、成本偏差分析以及成本纠偏措施。

### 二、核心讲解

#### 1. 成本监控方法

**成本监控方式**：

| 监控方式 | 说明 | 频率 |
|----------|------|------|
| **日常监控** | 日常费用支出监控 | 每日 |
| **定期检查** | 阶段性成本检查 | 每月/每季度 |
| **专项检查** | 针对特定支出检查 | 按需 |
| **审计监控** | 财务审计 | 项目结束 |

**成本监控方法**：

| 方法 | 说明 |
|------|------|
| **成本报告** | 通过报告了解成本情况 |
| **费用审核** | 审核各项费用支出 |
| **预算对比** | 对比实际支出与预算 |
| **价值分析** | 分析投入与产出的价值 |

#### 2. 成本偏差分析

**挣值管理（EVM）基本概念**：

| 概念 | 英文 | 说明 | 计算公式 |
|------|------|------|----------|
| **计划价值** | PV（Planned Value） | 计划完成工作的预算价值 | - |
| **实际成本** | AC（Actual Cost） | 实际完成工作的成本 | - |
| **挣值** | EV（Earned Value） | 实际完成工作的预算价值 | - |
| **成本偏差** | CV（Cost Variance） | 成本偏差 | CV = EV - AC |
| **进度偏差** | SV（Schedule Variance） | 进度偏差 | SV = EV - PV |
| **成本绩效指数** | CPI（Cost Performance Index） | 成本绩效 | CPI = EV / AC |
| **进度绩效指数** | SPI（Schedule Performance Index） | 进度绩效 | SPI = EV / PV |

**偏差分析判断**：

| 指标 | 判断标准 | 含义 |
|------|----------|------|
| CV > 0 | 成本低于预算 | 节约 |
| CV < 0 | 成本超出预算 | 超支 |
| SV > 0 | 进度超前 | 提前 |
| SV < 0 | 进度滞后 | 延迟 |
| CPI > 1 | 成本绩效好 | 节约 |
| CPI < 1 | 成本绩效差 | 超支 |

#### 3. 成本纠偏措施

**成本纠偏措施**：

| 措施 | 说明 |
|------|------|
| **节约材料** | 优化材料使用、减少浪费 |
| **提高效率** | 提高人员工作效率 |
| **优化方案** | 采用更经济的方案 |
| **变更范围** | 适当减少非必要功能 |
| **索赔补偿** | 向责任方索赔 |

### 三、通俗例子

**挣值管理例子**：
就像做预算吃饭：
- 计划中午吃30元（PV=30）
- 实际花了25元（AC=25）
- 实际吃了28元的东西（EV=28）
- 成本偏差 CV = 28-25 = +3元（节约3元）
- 进度偏差 SV = 28-30 = -2元（进度稍慢）

**成本纠偏例子**：
就像本月生活费超支了：
- 少下馆子（节约材料）
- 自己做饭（提高效率）
- 买便宜点的菜（优化方案）
- 减少娱乐开支（变更范围）

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **挣值管理** | PV、AC、EV、CV、SV、CPI、SPI | 七概念 |
| **成本偏差** | CV = EV - AC | 成本偏差公式 |
| **进度偏差** | SV = EV - PV | 进度偏差公式 |
| **纠偏措施** | 节约、提高、优化、变更、索赔 | 五措施 |

## 10.5 进度与成本综合管理

### 一、教学目标

掌握进度与成本的关系，以及如何进行综合管理和优化。

### 二、核心讲解

#### 1. 进度与成本的关系

**进度-成本关系曲线**：

```
成本
  │
  │    ╭───╮
  │   ╱     ╲         ← 直接成本随进度加快而增加
  │  ╱       ╲
  │ ╱         ╲
  │╱           ╲
  └─────────────→ 进度
     正常  加快
```

**进度与成本的权衡**：

| 权衡情况 | 进度加快 | 进度放缓 |
|----------|----------|----------|
| **直接成本** | 增加 | 减少 |
| **间接成本** | 减少 | 增加 |
| **总成本** | 可能增加或减少 | 可能增加或减少 |

#### 2. 综合优化方法

**项目优化目标**：

| 优化类型 | 目标 | 方法 |
|----------|------|------|
| **工期优化** | 在成本最低的前提下缩短工期 | 赶工、快速跟进 |
| **成本优化** | 在工期最短的前提下降低成本 | 资源优化、方案优化 |
| **资源优化** | 在满足约束的条件下优化资源使用 | 资源平衡、资源平滑 |

**优化原则**：
- 优先优化关键路径上的任务
- 选择成本最低的优化方案
- 确保优化后风险可控

### 三、通俗例子

**进度-成本权衡例子**：
就像赶飞机：
- 打车去机场（进度快但成本高）
- 坐公交去机场（进度慢但成本低）
- 需要权衡：错过飞机损失更大，还是打车费更贵？

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **进度-成本关系** | 进度加快直接成本增、间接成本减 | 权衡关系 |
| **综合优化类型** | 工期优化、成本优化、资源优化 | 三优化 |
| **优化原则** | 关键路径优先、成本最低、风险可控 | 三原则 |

### 二、易混淆概念辨析

#### 1. 进度偏差 vs 成本偏差

| 概念 | 公式 | 大于0含义 | 小于0含义 |
|------|------|-----------|-----------|
| **进度偏差 SV** | EV - PV | 进度超前 | 进度滞后 |
| **成本偏差 CV** | EV - AC | 成本节约 | 成本超支 |

#### 2. CPI vs SPI

| 概念 | 公式 | 大于1含义 | 小于1含义 |
|------|------|-----------|-----------|
| **成本绩效指数 CPI** | EV / AC | 成本绩效好 | 成本绩效差 |
| **进度绩效指数 SPI** | EV / PV | 进度绩效好 | 进度绩效差 |

#### 3. 赶工 vs 快速跟进

| 概念 | 做法 | 风险 |
|------|------|------|
| **赶工** | 增加资源投入、加班 | 资源成本增加 |
| **快速跟进** | 并行进行原计划串行的任务 | 可能返工 |

### 三、记忆口诀汇总

| 知识点 | 记忆口诀 |
|----------|----------|
| 影响进度因素 | 人物技环管业 |
| 进度计划方法 | 甘关路径里程碑，网络计划四方法 |
| 进度控制措施 | 组技经信合 |
| 进度调整措施 | 赶并快缩重（赶工+并行+快速跟进+缩减范围+重新规划） |
| 成本类型 | 直间固变沉 |
| 成本估算方法 | 类参自下三专家 |
| 挣值管理概念 | 计划实际挣价值，偏差绩效六个数 |
| 进度成本权衡 | 进度加快直接增，间接减少要记清 |

### 四、真题演练

#### 题型一：计算分析题

**例题**：某项目计划价值PV=100万元，实际成本AC=90万元，挣值EV=95万元。计算CV、SV、CPI、SPI并判断项目状态。

**解析**：
- CV = EV - AC = 95 - 90 = +5万元（成本节约）
- SV = EV - PV = 95 - 100 = -5万元（进度滞后）
- CPI = EV / AC = 95 / 90 ≈ 1.056（成本绩效好）
- SPI = EV / PV = 95 / 100 = 0.95（进度绩效差）

#### 题型二：概念辨析题

**例题**：以下关于进度调整措施的说法，正确的是（ ）
A. 赶工会增加项目风险
B. 缩减范围需要业主同意
C. 快速跟进可以减少总成本
D. 并行作业会增加关键路径长度

**解析**：缩减范围涉及减少功能，必须经过业主同意。答案选B。

#### 题型三：判断推理题

**例题**：项目出现CV<0且SPI<1的情况，说明（ ）
A. 成本超支且进度滞后
B. 成本节约且进度超前
C. 成本超支且进度超前
D. 成本节约且进度滞后

**解析**：CV<0表示成本超支，SPI<1表示进度滞后。答案选A。

### 五、学习建议

1. **掌握公式**：重点掌握CV、SV、CPI、SPI的计算公式
2. **理解关系**：理解进度与成本的权衡关系
3. **对比记忆**：对比进度偏差和成本偏差的概念和判断标准
4. **多做练习**：通过计算题练习巩固挣值管理的计算
5. **联系实际**：将进度和成本知识与实际项目管理联系起来
