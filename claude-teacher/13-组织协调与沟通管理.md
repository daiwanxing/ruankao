# 第十三章 组织协调与沟通管理

## 13.1 组织协调概述

### 一、教学目标

掌握组织协调的基本概念、原则和方法。

### 二、核心讲解

#### 1. 组织协调基本概念

**组织协调定义**：在项目实施过程中，通过沟通、协商、调解等方式，处理各方关系，解决矛盾冲突，确保项目顺利进行的活动。

**组织协调目标**：
- 协调各方**利益平衡**
- 解决各方**矛盾冲突**
- 促进各方**协作配合**
- 确保项目**顺利实施**

#### 2. 协调范围

**需要协调的关系**：

| 关系类型 | 说明 | 协调内容 |
|----------|------|----------|
| **业主与承建单位** | 项目最主要的合同关系 | 合同履行、变更、付款 |
| **监理与业主** | 委托代理关系 | 工作汇报、问题请示 |
| **监理与承建单位** | 监理与被监理关系 | 质量监督、进度控制 |
| **承建单位之间** | 分包商之间的关系 | 界面协调、配合问题 |
| **与外部单位** | 政府、供应商等 | 审批、采购 |

#### 3. 协调原则

| 原则 | 说明 |
|------|------|
| **公正原则** | 公平公正处理各方关系 |
| **及时原则** | 及时发现问题、及时处理 |
| **协商原则** | 通过协商解决问题 |
| **一致原则** | 寻求各方都能接受的方案 |
| **可行原则** | 解决方案切实可行 |

### 三、通俗例子

**组织协调例子**：
就像组织一次团队活动：
- 要协调每个人的时间（协调人员）
- 要平衡每个人的需求（协调利益）
- 要解决出现的矛盾（解决冲突）
- 要确保活动顺利进行（确保目标）

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **协调目标** | 利益平衡、矛盾解决、协作配合、顺利实施 | 四目标 |
| **协调关系** | 业主-承建、监理-业主、监理-承建、分包、外部 | 五关系 |
| **协调原则** | 公正、及时、协商、一致、可行 | 五原则 |

## 13.2 沟通管理

### 一、教学目标

掌握沟通管理的基本概念、沟通方式和沟通技巧。

### 二、核心讲解

#### 1. 沟通基本概念

**沟通定义**：信息从发送者传递到接收者的过程。

**沟通要素**：

| 要素 | 说明 |
|------|------|
| **发送者** | 信息的发出方 |
| **接收者** | 信息的接收方 |
| **信息内容** | 传递的具体信息 |
| **沟通渠道** | 传递信息的途径 |
| **反馈** | 接收者的回应 |

**沟通类型**：

| 类型 | 说明 |
|------|------|
| **正式沟通** | 通过正式渠道的沟通 | 会议、报告、文件 |
| **非正式沟通** | 通过非正式渠道的沟通 | 聊天、电话 |
| **单向沟通** | 信息单向传递 | 通知、公告 |
| **双向沟通** | 信息双向交流 | 讨论、协商 |

#### 2. 沟通方式

**沟通方式类型**：

| 方式 | 说明 | 优点 | 缺点 |
|------|------|------|------|
| **书面沟通** | 通过书面形式沟通 | 正式、可追溯 | 速度慢 |
| **口头沟通** | 通过口头交流沟通 | 快速、灵活 | 不易追溯 |
| **会议沟通** | 通过会议进行沟通 | 多人参与、讨论深入 | 耗时 |
| **电子沟通** | 通过电子方式沟通 | 快速、便捷 | 缺乏感情 |

#### 3. 沟通技巧

**有效沟通技巧**：

| 技巧 | 说明 |
|------|------|
| **积极倾听** | 认真听取对方意见 |
| **明确表达** | 清晰表达自己的观点 |
| **及时反馈** | 及时给予回应 |
| **换位思考** | 站在对方角度考虑 |
| **控制情绪** | 保持冷静理性 |

**沟通障碍及克服**：

| 障碍 | 克服方法 |
|------|----------|
| **语言障碍** | 使用通俗易懂的语言 |
| **文化障碍** | 尊重不同文化背景 |
| **心理障碍** | 建立信任关系 |
| **技术障碍** | 选择合适的沟通方式 |
| **层级障碍** | 减少沟通层级 |

### 三、通俗例子

**沟通要素例子**：
就像发微信：
- 你发消息（发送者）
- 对方收到消息（接收者）
- 你发的内容（信息内容）
- 通过微信发送（沟通渠道）
- 对方回复（反馈）

**沟通技巧例子**：
就像跟外国朋友聊天：
- 听对方说话时认真听（积极倾听）
- 用简单的词说话（明确表达）
- 对方说完你回复（及时反馈）
- 想一下对方为什么这么说（换位思考）
- 不要急躁（控制情绪）

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **沟通要素** | 发送者、接收者、内容、渠道、反馈 | 五要素 |
| **沟通类型** | 正式、非正式、单向、双向 | 四类型 |
| **沟通方式** | 书面、口头、会议、电子 | 四方式 |
| **沟通技巧** | 倾听、表达、反馈、换位、控制 | 五技巧 |
| **沟通障碍** | 语言、文化、心理、技术、层级 | 五障碍 |

## 13.3 会议管理

### 一、教学目标

掌握项目会议的类型、组织方法和会议纪要编写。

### 二、核心讲解

#### 1. 项目会议类型

**项目会议类型**：

| 会议类型 | 说明 | 参与人员 | 频率 |
|----------|------|----------|------|
| **项目启动会** | 项目启动时的会议 | 各方负责人 | 项目开始 |
| **例会** | 定期召开的工作例会 | 项目团队 | 每周/每月 |
| **专题会** | 针对特定问题的会议 | 相关人员 | 按需 |
| **协调会** | 协调各方关系的会议 | 各方代表 | 按需 |
| **验收会** | 项目验收时的会议 | 各方负责人 | 验收时 |

#### 2. 会议组织

**会议组织流程**：

| 阶段 | 工作内容 |
|------|----------|
| **会前准备** | 确定议题、通知参会人员、准备资料 |
| **会议进行** | 主持人引导、控制议程、记录要点 |
| **会后跟进** | 整理纪要、跟踪落实、反馈结果 |

**会议组织要点**：

| 要点 | 说明 |
|------|------|
| **明确议题** | 会议要有明确的议题 |
| **控制时间** | 控制会议时间，不拖延 |
| **充分讨论** | 给予充分的讨论时间 |
| **形成结论** | 会议要有明确结论 |
| **跟踪落实** | 跟踪会议决定事项 |

#### 3. 会议纪要

**会议纪要内容**：

| 内容 | 说明 |
|------|------|
| **会议基本信息** | 会议名称、时间、地点、参会人员 |
| **会议议题** | 会议讨论的议题 |
| **讨论内容** | 各议题的讨论情况 |
| **形成决议** | 会议形成的决定 |
| **待办事项** | 需要后续办理的事项 |
| **下次会议** | 下次会议的安排 |

**会议纪要编写要求**：

| 要求 | 说明 |
|------|------|
| **内容准确** | 内容要准确反映会议情况 |
| **条理清晰** | 条理要清晰分明 |
| **语言规范** | 语言要规范正式 |
| **及时编写** | 要及时编写和分发 |
| **存档备查** | 要存档备查 |

### 三、通俗例子

**会议组织例子**：
就像组织一次同学聚会：
- 会前：确定时间地点、通知同学、准备场地
- 会中：控制节奏、活跃气氛、拍照留念
- 会后：整理照片、发群里、约定下次

### 四、知识点速记

## 13.4 冲突管理

### 一、教学目标

掌握冲突的原因、类型和处理方法。

### 二、核心讲解

#### 1. 冲突概述

**冲突定义**：在项目实施过程中，各方因利益、观点、需求等不同而产生的矛盾和分歧。

**冲突产生原因**：

| 原因类型 | 具体原因 |
|----------|----------|
| **利益冲突** | 各方利益诉求不同 |
| **资源冲突** | 资源分配不均 |
| **进度冲突** | 进度要求不一致 |
| **技术冲突** | 技术方案有分歧 |
| **沟通冲突** | 沟通不畅产生误解 |
| **权力冲突** | 权力范围不清 |

#### 2. 冲突类型

**冲突类型分类**：

| 分类标准 | 冲突类型 | 说明 |
|----------|----------|------|
| **按冲突程度** | 轻微冲突 | 小分歧，可忽略 |
|  | 一般冲突 | 需要处理的分歧 |
|  | 严重冲突 | 重大矛盾，需解决 |
| **按冲突性质** | 建设性冲突 | 有利于项目发展 |
|  | 破坏性冲突 | 影响项目进行 |

#### 3. 冲突处理

**冲突处理策略**：

| 策略 | 说明 | 适用情况 |
|------|------|----------|
| **回避** | 暂时搁置冲突 | 冲突不重要 |
| **迁就** | 满足对方要求 | 维持关系更重要 |
| **竞争** | 坚持自己立场 | 己方立场正确 |
| **妥协** | 双方各让一步 | 双方都有道理 |
| **合作** | 寻求双赢方案 | 双方利益都重要 |

**冲突处理步骤**：

```
识别冲突 → 分析原因 → 制定方案 → 沟通协商 → 达成共识 → 跟踪反馈
```

### 三、通俗例子

**冲突处理例子**：
就像两个人争着付账：
- 回避：说"下次再说吧"
- 迁就："那你来付吧"
- 竞争："今天必须我付"
- 妥协："一人付一半"
- 合作："这次你付，下次我付"

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **冲突原因** | 利益、资源、进度、技术、沟通、权力 | 六原因 |
| **冲突类型** | 轻微、一般、严重、建设性、破坏性 | 五类型 |
| **处理策略** | 回避、迁就、竞争、妥协、合作 | 五策略 |
| **处理步骤** | 识别、分析、制定、沟通、达成、跟踪 | 六步骤 |

### 四、真题演练

#### 题型一：策略选择题

**例题**：在冲突处理中，当冲突双方的利益都很重要，且时间充裕时，应采用（ ）策略
A. 回避
B. 迁就
C. 竞争
D. 合作

**解析**：当双方利益都重要且有时间时，应采用合作策略，寻求双赢方案。答案选D。

#### 题型二：概念辨析题

**例题**：以下关于建设性冲突和破坏性冲突的说法，正确的是（ ）
A. 建设性冲突会阻碍项目进行
B. 破坏性冲突有利于项目发展
C. 建设性冲突可以激发创新
D. 两种冲突没有区别

**解析**：建设性冲突有利于项目发展，可以激发创新和改进方案。答案选C。

#### 题型三：分类记忆题

**例题**：会议纪要的内容不包括（ ）
A. 会议基本信息
B. 会议议题
C. 参会人员的个人隐私
D. 会议决议

**解析**：会议纪要应包含基本信息、议题、讨论、决议、待办、下次会议等内容，不包括个人隐私。答案选C。
