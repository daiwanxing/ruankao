# 第十一章 合同管理与知识产权

## 11.1 合同管理概述

### 一、教学目标

掌握合同的基本概念、合同的订立与履行、合同变更与索赔管理。

### 二、核心讲解

#### 1. 合同基本概念

**合同定义**：平等主体的自然人、法人、其他组织之间设立、变更、终止民事权利义务关系的协议。

**合同要素**：

| 要素 | 说明 | 举例 |
|------|------|------|
| **主体** | 签订合同的双方或多当事人 | 业主单位、承建单位 |
| **客体** | 合同权利义务指向的对象 | 信息系统工程 |
| **内容** | 合同规定的权利和义务 | 工程范围、工期、费用 |

**合同类型**：

| 分类标准 | 合同类型 | 说明 |
|----------|----------|------|
| **按内容** | 总承包合同 | 将整个项目承包给一方 |
|  | 分包合同 | 总承包商将部分工作分包 |
|  | 劳务合同 | 提供劳务服务的合同 |
| **按计价方式** | 总价合同 | 约定固定总价 |
|  | 单价合同 | 按单价计价 |
|  | 成本加酬金合同 | 实报实销加酬金 |
| **按形式** | 书面合同 | 以书面形式签订 |
|  | 口头合同 | 以口头形式约定 |

#### 2. 合同订立

**合同订立程序**：

| 阶段 | 说明 |
|------|------|
| **要约** | 一方提出订立合同的建议 |
| **承诺** | 另一方接受要约 |
| **签约** | 双方签订合同 |

**合同订立原则**：

| 原则 | 说明 |
|------|------|
| **平等原则** | 合同双方地位平等 |
| **自愿原则** | 合同内容由双方自愿约定 |
| **公平原则** | 合同内容公平合理 |
| **诚实信用原则** | 双方诚实守信 |
| **合法原则** | 合同内容符合法律规定 |

#### 3. 合同履行

**合同履行原则**：

| 原则 | 说明 |
|------|------|
| **全面履行** | 按照合同约定全面履行义务 |
| **诚实履行** | 诚实守信履行合同 |
| **协作履行** | 双方相互配合履行 |

**合同履行内容**：

| 履行内容 | 说明 |
|----------|------|
| **质量履行** | 按约定质量标准交付 |
| **数量履行** | 按约定数量交付 |
| **时间履行** | 按约定时间交付 |
| **地点履行** | 按约定地点交付 |
| **方式履行** | 按约定方式交付 |

### 三、通俗例子

**合同要素例子**：
就像租房合同：
- 主体：房东和你
- 客体：这套房子
- 内容：租金多少、租多久、押金多少

**要约与承诺例子**：
就像买东西：
- 你问"这个多少钱"（要约邀请）
- 老板说"100块"（要约）
- 你说"好，拿一个"（承诺）
- 成交！

## 11.2 合同变更管理

### 一、教学目标

掌握合同变更的原因、程序和处理方法。

### 二、核心讲解

#### 1. 合同变更概述

**合同变更定义**：在合同成立后，当事人在原合同的基础上对合同的内容进行修改或补充。

**合同变更原因**：

| 原因类型 | 具体原因 |
|----------|----------|
| **需求变更** | 业主需求发生变化 |
| **技术变更** | 技术方案需要调整 |
| **环境变更** | 外部环境发生变化 |
| **法规变更** | 相关法律法规变化 |
| **不可抗力** | 不可预见的事件发生 |

**合同变更特点**：
- 合同变更需经双方协商一致
- 合同变更应采用书面形式
- 合同变更不影响已履行部分

#### 2. 合同变更程序

**合同变更流程**：

```
变更提出 → 变更评估 → 变更审批 → 变更实施 → 变更记录
    ↓          ↓          ↓          ↓          ↓
  提出变更  评估影响  领导审批   执行变更   记录归档
```

**变更处理原则**：
| 原则 | 说明 |
|------|------|
| **及时处理** | 及时处理变更，避免影响项目 |
| **书面确认** | 变更需书面确认 |
| **影响评估** | 评估变更对进度、成本、质量的影响 |
| **文档更新** | 及时更新相关文档 |

#### 3. 合同变更控制

**变更控制措施**：

| 措施 | 说明 |
|------|------|
| **变更申请** | 提交正式的变更申请 |
| **变更审核** | 审核变更的必要性和合理性 |
| **变更批准** | 授权人批准变更 |
| **变更实施** | 按照批准的变更实施 |
| **变更验证** | 验证变更实施效果 |

### 三、通俗例子

**合同变更例子**：
就像装修房子签了合同，后来想要加一个柜子：
- 提出变更：跟装修公司说要加柜子
- 评估影响：装修公司评估要多花多少钱、多少时间
- 协商价格：双方协商加柜子的费用
- 签订补充协议：签变更合同
- 执行变更：装修公司做柜子

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **变更原因** | 需求、技术、环境、法规、不可抗力 | 五原因 |
| **变更流程** | 提出、评估、审批、实施、记录 | 五步骤 |
| **变更控制** | 申请、审核、批准、实施、验证 | 五控制 |

## 11.3 索赔管理

### 一、教学目标

掌握索赔的定义、类型、程序和处理方法。

### 二、核心讲解

#### 1. 索赔基本概念

**索赔定义**：在合同履行过程中，一方因另一方未履行合同义务或履行不符合约定而造成的损失，向对方提出赔偿的要求。

**索赔类型**：

| 分类标准 | 索赔类型 | 说明 |
|----------|----------|------|
| **按索赔方** | 业主索赔 | 业主向承建单位索赔 |
|  | 承建单位索赔 | 承建单位向业主索赔 |
| **按索赔内容** | 费用索赔 | 索赔费用损失 |
|  | 工期索赔 | 索赔延长工期 |
|  | 费用加工期索赔 | 索赔费用和工期 |
| **按索赔依据** | 合同索赔 | 依据合同条款索赔 |
|  | 法规索赔 | 依据法律法规索赔 |

#### 2. 索赔程序

**索赔流程**：

```
索赔提出 → 索赔审核 → 索赔协商 → 索赔裁决 → 索赔执行
    ↓          ↓          ↓          ↓          ↓
  提出索赔  审核索赔  双方协商   第三方裁决  执行赔偿
```

**索赔时限**：
- 索赔应在事件发生后**28天内**提出
- 索赔报告应在提出后**14天内**提交

#### 3. 索赔处理

**索赔证据**：

| 证据类型 | 具体内容 |
|----------|----------|
| **合同文件** | 合同、协议、招标文件 |
| **来往函件** | 双方往来的函件 |
| **会议记录** | 会议纪要、谈判记录 |
| **工程资料** | 进度计划、工程日志 |
| **财务资料** | 发票、收据、账单 |

**索赔金额计算**：

| 计算方法 | 说明 |
|----------|------|
| **实际费用法** | 按实际发生的费用计算 |
| **总费用法** | 按实际总费用与计划费用的差额计算 |
| **修正总费用法** | 修正总费用后计算差额 |

### 三、通俗例子

**索赔例子**：
就像订外卖：
- 外卖超时1小时送到（违约）
- 你要求赔偿（索赔）
- 外卖平台核实后给你退了钱（索赔处理）

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **索赔类型** | 费用、工期、费用加工期 | 三类型 |
| **索赔流程** | 提出、审核、协商、裁决、执行 | 五步骤 |
| **索赔时限** | 事件后28天提出，14天内提交报告 | 28/14 |
| **索赔证据** | 合同、函件、会议、资料、财务 | 五证据 |

## 11.4 知识产权管理

### 一、教学目标

掌握知识产权的基本概念、类型和保护方法。

### 二、核心讲解

#### 1. 知识产权基本概念

**知识产权定义**：人们对其智力劳动成果依法享有的专有权利。

**知识产权特征**：

| 特征 | 说明 |
|------|------|
| **无形性** | 没有实体形态 |
| **专有性** | 权利人独占使用 |
| **地域性** | 在授予国范围内有效 |
| **时间性** | 有保护期限限制 |

**知识产权类型**：

| 类型 | 保护内容 | 保护期限 |
|------|----------|----------|
| **著作权** | 文学、艺术、科学作品 | 作者终生+死后50年 |
| **专利权** | 发明、实用新型、外观设计 | 20年/10年/15年 |
| **商标权** | 商品或服务的商标 | 10年，可续展 |
| **商业秘密** | 不为公众所知的信息 | 无期限限制 |

#### 2. 知识产权保护

**知识产权保护措施**：

| 措施 | 说明 |
|------|------|
| **法律保护** | 通过法律手段保护 |
| **合同保护** | 通过合同约定保护 |
| **技术保护** | 采用技术手段保护 |
| **管理保护** | 建立管理制度保护 |

**知识产权归属**：

| 成果类型 | 归属原则 |
|----------|----------|
| **职务作品** | 一般归单位所有 |
| **委托作品** | 按合同约定归属 |
| **合作作品** | 合作各方共同所有 |

#### 3. 侵权与处罚

**侵权行为**：

| 侵权类型 | 说明 |
|----------|------|
| **著作权侵权** | 未经授权复制、传播作品 |
| **专利权侵权** | 未经授权使用专利 |
| **商标权侵权** | 未经授权使用商标 |

**处罚措施**：

| 处罚类型 | 说明 |
|----------|------|
| **民事责任** | 停止侵害、赔偿损失 |
| **行政责任** | 罚款、没收违法所得 |
| **刑事责任** | 构成犯罪的追究刑事责任 |

### 三、通俗例子

**知识产权例子**：
就像写小说：
- 你写的小说有著作权（别人不能随便抄）
- 你给小说起的名字可以注册商标（别人不能用这个名字）
- 如果你有发明创造可以申请专利（别人不能生产同样的东西）

**职务作品例子**：
就像在公司上班写的代码，一般来说代码属于公司，而不是你自己。

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **知识产权特征** | 无形性、专有性、地域性、时间性 | 四特征 |
| **知识产权类型** | 著作权、专利权、商标权、商业秘密 | 四类型 |
| **保护措施** | 法律、合同、技术、管理 | 四措施 |
| **侵权类型** | 著作权侵权、专利权侵权、商标权侵权 | 三侵权 |
| **处罚措施** | 民事责任、行政责任、刑事责任 | 三责任 |

### 二、易混淆概念辨析

#### 1. 要约 vs 承诺

| 概念 | 定义 | 特点 |
|------|------|------|
| **要约** | 一方提出订立合同的建议 | 可撤回、撤销 |
| **承诺** | 另一方接受要约 | 不得撤回 |

#### 2. 著作权 vs 专利权 vs 商标权

| 权利 | 保护对象 | 保护条件 | 保护期限 |
|------|----------|----------|----------|
| **著作权** | 作品 | 自动产生 | 终生+50年 |
| **专利权** | 发明创造 | 申请审批 | 20年/10年/15年 |
| **商标权** | 商标 | 注册取得 | 10年，可续展 |

#### 3. 合同变更 vs 合同转让

| 概念 | 定义 | 区别 |
|------|------|------|
| **合同变更** | 修改或补充合同内容 | 改变内容 |
| **合同转让** | 将合同权利义务转让给第三方 | 改变主体 |

### 三、记忆口诀汇总

| 知识点 | 记忆口诀 |
|----------|----------|
| 合同三要素 | 主客内（主体+客体+内容） |
| 合同计价方式 | 总单成（总价+单价+成本加酬金） |
| 合同订立程序 | 要承签（要约+承诺+签约） |
| 合同变更流程 | 提评审施记（提出+评估+审批+实施+记录） |
| 索赔类型 | 费工期（费用+工期+费用工期） |
| 索赔时限 | 二八十四（28天提出，14天提交报告） |

### 四、真题演练

#### 题型一：概念辨析题

**例题**：以下关于要约和承诺的说法，正确的是（ ）
A. 承诺可以撤回
B. 要约一经发出不可撤回
C. 承诺是对要约的接受
D. 要约和承诺可以同时进行

**解析**：承诺是对要约的接受，一旦做出不得撤回；要约可以撤回或撤销。答案选C。

#### 题型二：计算判断题

**例题**：某事件发生后35天提出索赔，是否有效？
A. 有效
B. 无效，因为超过28天
C. 无效，因为超过30天
D. 视情况而定

**解析**：索赔应在事件发生后28天内提出，35天已超过时限，通常无效。答案选B。

#### 题型三：分类记忆题

**例题**：著作权的保护期限是（ ）
A. 10年
B. 20年
C. 作者终生+死后50年
D. 无期限限制

**解析**：著作权的保护期限是作者终生加死后50年。答案选C。

### 五、学习建议

1. **理解合同流程**：理解从订立到履行、变更、索赔的完整流程
2. **区分知识产权类型**：区分著作权、专利权、商标权的保护对象和期限
3. **掌握时间节点**：记住索赔28天、14天等关键时间节点
4. **对比学习**：对比要约与承诺、变更与转让等易混概念
5. **联系实际**：将合同知识与实际签订合同经历联系起来
