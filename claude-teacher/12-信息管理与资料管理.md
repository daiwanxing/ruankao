# 第十二章 信息管理与资料管理

## 12.1 信息管理概述

### 一、教学目标

掌握信息管理的基本概念、信息分类和信息管理方法。

### 二、核心讲解

#### 1. 信息管理基本概念

**信息定义**：经过加工处理后对决策有价值的数据。

**信息管理定义**：对信息的收集、存储、处理、传输、应用等过程的管理。

**信息管理目标**：
- 确保信息的**准确性**
- 确保信息的**及时性**
- 确保信息的**安全性**
- 确保信息的**可用性**

#### 2. 信息分类

**按信息性质分类**：

| 类型 | 说明 | 举例 |
|------|------|------|
| **技术信息** | 技术相关的信息 | 设计文档、技术方案 |
| **管理信息** | 管理相关的信息 | 进度报告、质量报告 |
| **经济信息** | 经济和财务信息 | 成本数据、付款记录 |
| **合同信息** | 合同相关信息 | 合同文本、变更记录 |

**按信息来源分类**：

| 类型 | 说明 | 举例 |
|------|------|------|
| **内部信息** | 项目内部产生的信息 | 会议纪要、工作日志 |
| **外部信息** | 来自外部的信息 | 业主通知、供应商资料 |

**按信息形式分类**：

| 类型 | 说明 |
|------|------|
| **文字信息** | 以文字形式记录的信息 |
| **数据信息** | 以数据形式记录的信息 |
| **图形信息** | 以图形、图表形式记录的信息 |
| **多媒体信息** | 以音频、视频等形式记录的信息 |

#### 3. 信息管理方法

**信息收集方法**：

| 方法 | 说明 |
|------|------|
| **调查收集** | 通过调查获取信息 |
| **会议收集** | 通过会议获取信息 |
| **报告收集** | 通过报告获取信息 |
| **现场收集** | 通过现场检查获取信息 |

**信息处理方法**：

| 方法 | 说明 |
|------|------|
| **分类整理** | 将信息按类别整理 |
| **分析加工** | 对信息进行分析处理 |
| **存储归档** | 将信息存储归档 |
| **传递分发** | 将信息传递给需要的人 |

### 三、通俗例子

**信息管理例子**：
就像管理一个图书馆的书：
- 收集：买新书或接受捐赠
- 分类：按类别分门别类
- 存储：把书放到书架上
- 检索：方便找到需要的书
- 借阅：把书借给需要的人

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **信息管理目标** | 准确、及时、安全、可用 | 四目标 |
| **信息分类** | 技术、管理、经济、合同 | 四类型 |
| **信息来源** | 内部信息、外部信息 | 两来源 |
| **信息形式** | 文字、数据、图形、多媒体 | 四形式 |
| **信息收集** | 调查、会议、报告、现场 | 四方法 |

## 12.2 监理信息管理

### 一、教学目标

掌握监理信息管理的内容、方法和监理信息流。

### 二、核心讲解

#### 1. 监理信息内容

**监理信息类型**：

| 类型 | 说明 | 举例 |
|------|------|------|
| **监理规划信息** | 监理规划和实施细则 | 监理规划、监理细则 |
| **监理过程信息** | 监理工作过程信息 | 监理日志、旁站记录 |
| **监理成果信息** | 监理工作成果信息 | 监理报告、评估报告 |
| **监理协调信息** | 监理协调工作信息 | 协调记录、会议纪要 |

**监理信息要素**：

| 要素 | 说明 |
|------|------|
| **信息内容** | 信息具体是什么 |
| **信息来源** | 信息来自哪里 |
| **信息时间** | 信息发生的时间 |
| **信息地点** | 信息发生的地点 |
| **信息责任人** | 谁对信息负责 |

#### 2. 监理信息流

**信息流向**：

| 流向类型 | 说明 |
|----------|------|
| **自上而下** | 从高层到低层的信息传递 |
| **自下而上** | 从低层到高层的信息报告 |
| **横向流动** | 同级之间的信息交流 |
| **外部流动** | 与外部单位的信息交流 |

**信息传递方式**：

| 方式 | 说明 | 适用场景 |
|------|------|----------|
| **书面传递** | 通过书面文件传递 | 正式信息 |
| **会议传递** | 通过会议传递 | 需要讨论的信息 |
| **口头传递** | 通过口头传达 | 简单信息 |
| **电子传递** | 通过电子邮件等传递 | 紧急信息 |

#### 3. 监理信息系统

**监理信息系统功能**：

| 功能 | 说明 |
|------|------|
| **信息收集** | 收集各类监理信息 |
| **信息存储** | 存储监理信息 |
| **信息检索** | 方便查找信息 |
| **信息统计** | 统计监理工作数据 |
| **信息报表** | 生成监理报表 |

### 三、通俗例子

**监理信息流例子**：
就像公司的信息传递：
- 总经理的命令传到部门经理（自上而下）
- 员工的工作报告给部门经理（自下而上）
- 部门经理之间交流（横向流动）
- 公司与客户沟通（外部流动）

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **监理信息类型** | 规划、过程、成果、协调 | 四类型 |
| **监理信息要素** | 内容、来源、时间、地点、责任人 | 五要素 |
| **信息流向** | 自上而下、自下而上、横向、外部 | 四流向 |
| **信息传递方式** | 书面、会议、口头、电子 | 四方式 |
| **信息系统功能** | 收集、存储、检索、统计、报表 | 五功能 |

## 12.3 资料管理

### 一、教学目标

掌握监理资料管理的内容、分类和归档要求。

### 二、核心讲解

#### 1. 资料管理概述

**资料定义**：在监理工作中形成的各类文档、记录、报告等。

**资料管理目标**：
- 确保资料的**完整性**
- 确保资料的**准确性**
- 确保资料的**可追溯性**
- 确保资料的**安全性**

#### 2. 资料分类

**按资料性质分类**：

| 类型 | 说明 | 举例 |
|------|------|------|
| **管理类资料** | 管理相关的资料 | 管理制度、工作计划 |
| **技术类资料** | 技术相关的资料 | 设计文件、测试报告 |
| **监理类资料** | 监理独有的资料 | 监理日志、旁站记录 |
| **合同类资料** | 合同相关的资料 | 合同文本、变更记录 |

**按资料阶段分类**：

| 阶段 | 资料类型 |
|------|----------|
| **设计阶段** | 设计评审记录、设计变更文件 |
| **实施阶段** | 监理日志、旁站记录、进度报告 |
| **验收阶段** | 验收报告、评估报告、移交记录 |

#### 3. 资料编制要求

**资料编制原则**：

| 原则 | 说明 |
|------|------|
| **真实性** | 内容真实准确 |
| **完整性** | 内容完整不缺项 |
| **规范性** | 格式规范统一 |
| **及时性** | 及时编制和归档 |

**资料编制要求**：

| 要求 | 说明 |
|------|------|
| **用词规范** | 使用规范的用语 |
| **格式统一** | 采用统一的格式 |
| **签署齐全** | 相关人员签字齐全 |
| **日期准确** | 标注准确的日期 |

#### 4. 资料归档

**归档要求**：

| 要求 | 说明 |
|------|------|
| **分类归档** | 按类别分类存放 |
| **编制目录** | 编制资料目录 |
| **编号管理** | 统一编号管理 |
| **定期归档** | 定期进行归档 |

**归档期限**：

| 资料类型 | 归档期限 |
|----------|----------|
| **永久保存资料** | 长期保存 |
| **长期保存资料** | 保存15年以上 |
| **短期保存资料** | 保存5年以上 |

### 三、通俗例子

**资料管理例子**：
就像管理自己的档案：
- 上学时的成绩单、工作后的合同（分类保存）
- 按时间顺序整理（编制目录）
- 放在安全的地方（编号管理）
- 重要文件多留几份（备份归档）

### 四、知识点速记

| 知识点 | 关键要点 | 记忆口诀 |
|--------|----------|----------|
| **资料管理目标** | 完整性、准确性、可追溯性、安全性 | 四目标 |
| **资料分类** | 管理、技术、监理、合同 | 四类型 |
| **编制原则** | 真实性、完整性、规范性、及时性 | 四原则 |
| **归档要求** | 分类、目录、编号、定期 | 四要求 |
| **归档期限** | 永久、长期、短期 | 三期限 |

| 概念 | 特点 | 时机 |

| **监理报告** | 分析性、总结性 | 按阶段或需要 |

| 期限 | 保存时间 | 适用资料 |

| **永久** | 长期保存 | 重要档案 |

| **短期** | 5年以上 | 临时档案 |

### 四、真题演练

#### 题型一：概念辨析题

**例题**：以下关于内部信息和外部信息的说法，正确的是（ ）
A. 内部信息比外部信息重要
B. 外部信息不需要管理
C. 内部信息来自项目内部，外部信息来自外部单位
D. 内部信息不需要收集

**解析**：内部信息来自项目内部产生，外部信息来自外部单位，两者都需要管理。答案选C。

#### 题型二：分类记忆题

**例题**：监理资料按性质分类不包括（ ）
A. 管理类资料
B. 技术类资料
C. 财务类资料
D. 监理类资料

**解析**：监理资料按性质分为管理、技术、监理、合同四类。答案选C。

#### 题型三：原则应用题

**例题**：资料编制应遵循的原则不包括（ ）
A. 真实性
B. 完整性
C. 随意性
D. 规范性

**解析**：资料编制原则是真实性、完整性、规范性、及时性，"随意性"不在其中。答案选C。

### 五、学习建议

1. **理解信息流**：理解信息的流向和传递方式
2. **掌握分类**：掌握信息分类和资料分类的标准
3. **重视编制**：重视资料编制和归档的要求
4. **区分报表报告**：区分监理报表和监理报告的不同
5. **多做练习**：通过练习题巩固知识点
