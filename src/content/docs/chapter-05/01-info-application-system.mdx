## 5.1 办公自动化系统（OA）

### 一、教学目标

掌握OA系统的基本概念、主要功能和实施要点。

### 二、核心讲解

#### 1. OA系统定义

办公自动化(Office Automation)是将现代化办公和计算机网络功能结合起来的一种新型的办公方式。

#### 2. OA系统主要功能

<MemoryCard 
  title="OA核心功能"
  formula="流程审批、文档管理、协同办公、信息发布"
  importance={3}
  category="key-point"
/>

| 功能模块 | 说明 |
|----------|------|
| **工作流管理** | 流程设计、流程审批 |
| **文档管理** | 文件存储、版本控制 |
| **通讯管理** | 邮件、即时通讯 |
| **知识管理** | 知识库、经验分享 |
| **信息发布** | 公告通知、新闻动态 |

## 5.2 企业资源计划（ERP）

### 一、核心讲解

#### 1. ERP系统定义

企业资源计划(Enterprise Resource Planning)是整合企业管理理念、业务流程、基础数据、人力物力、计算机硬件和软件于一体的企业资源管理系统。

#### 2. ERP核心模块

<MemoryCard 
  title="ERP三大核心"
  formula="财务管理、供应链管理、生产管理"
  importance={5}
  category="formula"
/>

| 模块 | 功能 |
|------|------|
| **财务管理** | 总账、应收应付、成本核算 |
| **供应链管理** | 采购、销售、库存 |
| **生产管理** | 计划、排程、制造执行 |
| **人力资源** | 人事、薪资、考勤 |

#### 3. ERP与MRP的关系

<Card title="MRP → MRPII → ERP演进" icon="information">
- **MRP**（物料需求计划）：仅物料管理
- **MRPII**（制造资源计划）：增加车间和制造管理
- **ERP**（企业资源计划）：整合企业所有资源
</Card>

## 5.3 客户关系管理（CRM）

### 一、核心讲解

#### 1. CRM系统定义

客户关系管理(Customer Relationship Management)以客户为中心，改善企业与客户之间关系的管理系统。

#### 2. CRM核心功能

| 功能 | 说明 |
|------|------|
| **客户信息管理** | 客户资料、联系历史 |
| **销售管理** | 销售机会、销售漏斗 |
| **营销管理** | 营销活动、市场分析 |
| **服务管理** | 客户服务、投诉处理 |

---

## 5.4 供应链管理（SCM）

### 一、核心讲解

#### 1. SCM系统定义

供应链管理(Supply Chain Management)对供应链中的物流、信息流、资金流进行计划、组织、协调和控制。

## 5.5 商业智能（BI）

### 一、核心讲解

#### 1. BI系统定义

商业智能(Business Intelligence)将企业中现有的数据转化为知识，帮助企业做出明智的业务经营决策。

#### 2. BI主要功能

| 功能 | 说明 |
|------|------|
| **数据采集** | 从各业务系统采集数据 |
| **数据分析** | OLAP多维分析 |
| **数据挖掘** | 发现数据中的规律 |
| **报表展现** | 可视化展示 |

### 一、核心讲解

#### 1. 电子商务模式

<MemoryCard 
  title="电商四大模式"

## 重点总结

### 本章必记考点 ⭐⭐⭐⭐

1. **OA核心**：流程审批、文档管理、协同办公、信息发布
2. **ERP三大核心**：财务管理、供应链管理、生产管理
3. **MRP演进**：MRP → MRPII → ERP
4. **CRM核心**：以客户为中心
5. **SCM覆盖**：供应商→制造商→分销商→零售商→客户
6. **BI功能**：数据采集→分析→挖掘→展现
7. **电商模式**：B2B、B2C、C2C、O2O
