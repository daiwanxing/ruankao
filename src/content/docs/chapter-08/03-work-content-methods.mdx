## 一、教学目标

掌握监理工作的具体内容、主要方法和工作流程。

## 二、核心讲解

### 1. 监理工作阶段划分

**监理工作贯穿项目全生命周期**：

| 项目阶段 | 监理工作内容 |
|----------|--------------|
| **项目决策阶段** | 可行性研究评审、项目建议书审核 |
| **设计阶段** | 设计方案评审、设计进度和质量管理 |
| **招标阶段** | 招标文件审查、招标过程监督 |
| **施工阶段** | 质量控制、进度控制、投资控制 |
| **验收阶段** | 验收测试组织、验收文档审核 |
| **运维阶段** | 运维服务质量监督 |

### 2. 监理工作方法

| 方法 | 说明 | 适用场景 |
|------|------|----------|
| **审核** | 审核各类文档、方案、报告 | 设计方案、进度计划等 |
| **检查** | 现场检查、巡视检查 | 工程质量、安全措施 |
| **试验** | 各类测试、检测 | 软件测试、设备检测 |
| **旁站** | 现场全程监督关键工序 | 隐蔽工程、关键节点 |
| **签认** | 对重要事项签字确认 | 验收签字、付款签认 |
| **协调** | 协调各方关系和争议 | 解决纠纷、统一意见 |

### 3. 监理主要工作

#### (1) 质量控制

| 控制阶段 | 控制内容 |
|----------|----------|
| **事前控制** | 审核承建单位资质、检查质量保证体系 |
| **事中控制** | 巡视检查、旁站监理、平行检验 |
| **事后控制** | 验收测试、质量评估 |

#### (2) 进度控制

| 控制类型 | 具体措施 |
|----------|----------|
| **进度计划审核** | 审核项目进度计划的合理性和可行性 |
| **进度跟踪** | 定期检查实际进度与计划进度的偏差 |
| **进度调整** | 分析偏差原因，督促采取纠偏措施 |
| **进度报告** | 编制进度监理报告，向业主汇报 |

#### (3) 投资控制

| 控制环节 | 具体措施 |
|----------|----------|
| **投资审核** | 审核工程量清单、工程变更 |
| **投资监控** | 跟踪投资使用情况 |
| **投资支付** | 审核工程款支付申请 |
| **投资结算** | 审核工程结算报告 |

### 4. 监理工作流程

```
监理工作流程：
项目承接 → 组建监理部 → 编制监理规划 → 实施监理工作 → 监理资料整理 → 项目总结
    ↓           ↓              ↓              ↓              ↓             ↓
  签订合同   人员配置      制定计划     日常监理      归档资料      经验总结
```

### 5. 监理三文件

监理工作涉及三个重要文件，它们在不同阶段编制，详细程度和作用各不相同。

#### (1) 监理大纲

**定义**：监理大纲是监理单位在投标阶段编制的文件，是投标文件的重要组成部分。

<MemoryCard
  title="监理大纲"
  formula="编制阶段：投标阶段 | 编制人：企业技术负责人 | 作用：争取中标"
  importance={4}
  category="key-point"
/>

| 特点 | 说明 |
|------|------|
| **编制时机** | 投标阶段，响应招标文件要求 |
| **编制主体** | 监理企业技术负责人主持编制 |
| **主要内容** | 监理工作目标、监理工作范围、监理方案、监理资源配置 |
| **作用** | 展示监理单位能力，争取中标 |

#### (2) 监理规划

**定义**：监理规划是在监理合同签订后、项目开始前编制的指导监理工作的纲领性文件。

<MemoryCard
  title="监理规划"
  formula="编制阶段：合同签订后 | 编制人：总监理工程师 | 作用：指导监理工作"
  importance={5}
  category="formula"
/>

| 特点 | 说明 |
|------|------|
| **编制时机** | 监理合同签订后，项目正式开始前 |
| **编制主体** | 总监理工程师主持编制 |
| **审批** | 监理单位技术负责人审批 |
| **主要内容** | 项目概况、监理工作依据、监理工作内容、监理工作方法及措施、监理工作制度、监理组织机构 |
| **作用** | 指导整个项目的监理工作 |

#### (3) 监理实施细则

**定义**：监理实施细则是在监理规划基础上，针对专业监理工作编制的更详细、更具体的操作文件。

<MemoryCard
  title="监理实施细则"
  formula="编制阶段：项目实施初期 | 编制人：专业监理工程师 | 审批：总监理工程师"
  importance={5}
  category="formula"
/>

| 特点 | 说明 |
|------|------|
| **编制时机** | 项目实施初期，收到设计文件后 |
| **编制主体** | 专业监理工程师编制 |
| **审批** | 总监理工程师审批 |
| **主要内容** | 专业工程特点、监理工作流程、监理工作要点、监理工作方法、具体的检查标准和频次 |
| **作用** | 指导具体专业的监理工作实施 |

#### (4) 三文件对比

| 对比维度 | 监理大纲 | 监理规划 | 监理实施细则 |
|----------|----------|----------|--------------|
| **编制阶段** | 投标阶段 | 合同签订后 | 实施初期 |
| **编制人** | 技术负责人 | 总监理工程师 | 专业监理工程师 |
| **详细程度** | 粗略 | 较详细 | 最详细、最具体 |
| **审批人** | - | 技术负责人 | 总监理工程师 |
| **作用** | 争取中标 | 指导监理工作 | 指导具体实施 |

## 三、通俗例子

<Card title="监理方法例子" icon="information">

- 审核（看作业）→ 检查（课堂巡视）→ 试验（考试）→ 旁站（监考）→ 签认（批改签字）
</Card>

<Card title="质量控制例子" icon="information">
就像做饭：
- 事前控制：检查食材是否新鲜
- 事中控制：监督烹饪过程
- 事后控制：品尝检查是否熟透
</Card>

## 六、重点总结

### 本节必记考点 ⭐⭐⭐⭐

1. **监理工作六方法**：审核、检查、试验、旁站、签认、协调
2. **质量控制三阶段**：事前控制（预防）、事中控制（监督）、事后控制（验收）
3. **进度控制四措施**：计划审核、跟踪、调整、报告
4. **监理工作流程**：项目承接→组建监理部→编制监理规划→实施监理工作→资料整理→项目总结
5. **监理三文件对比**：
   - 监理大纲：投标阶段、技术负责人编制、争取中标
   - 监理规划：合同签订后、总监理工程师编制、指导监理工作
   - 监理实施细则：实施初期、专业监理工程师编制、指导具体实施
6. **三文件编制人记忆**：大纲（技术负责人）、规划（总监）、细则（专业工程师）
