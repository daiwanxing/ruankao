## 一、教学目标

掌握监理组织的设置原则、监理人员的职责分工以及监理工程师的职业道德要求。

## 二、核心讲解

### 1. 监理组织设置原则

| 原则 | 含义 | 说明 |
|------|------|------|
| **专业分工** | 根据专业领域设置监理岗位 | 硬件、软件、网络、安全等 |
| **层次适当** | 监理组织层级合理 | 总监理工程师→总监代表→专业监理工程师→监理员 |
| **统一指挥** | 单一领导，避免多头指挥 | 确保指令统一 |
| **跨度合理** | 每个管理者管理人数适当 | 一般6-8人为宜 |
| **精干高效** | 人员精简，提高效率 | 避免人浮于事 |

### 2. 监理人员职责分工

**总监理工程师职责**：

| 职责 | 具体内容 |
|------|----------|
| 全面负责 | 负责监理项目的总体管理和协调 |
| 人员调配 | 调配监理人员和工作资源 |
| 审核批准 | 审批监理规划、实施细则 |
| 协调沟通 | 协调业主、承建单位关系 |
| 签发指令 | 签发开工令、停工令、复工令 |
| 争议处理 | 处理项目中的重大争议 |

**专业监理工程师职责**：

| 职责 | 具体内容 |
|------|----------|
| 专业负责 | 负责本专业领域的监理工作 |
| 技术指导 | 指导监理员开展具体工作 |
| 资料管理 | 编写本专业的监理资料 |
| 质量检查 | 检查本专业的工程质量 |
| 问题处理 | 处理本专业的一般问题 |

**监理员职责**：

| 职责 | 具体内容 |
|------|----------|
| 现场监理 | 在施工现场进行日常监理 |
| 记录工作 | 记录监理日志、旁站记录 |
| 问题报告 | 发现问题及时向上报告 |
| 资料收集 | 收集整理监理资料 |

### 3. 监理人员素质要求

**监理工程师应具备的素质**：

| 素质类型 | 具体要求 |
|----------|----------|
| **专业素质** | 扎实的信息系统专业知识 |
| **管理素质** | 具备项目管理能力 |
| **协调素质** | 良好的沟通协调能力 |
| **职业操守** | 公正、廉洁、诚实守信 |

### 4. 监理工程师职业道德

**监理工程师职业道德规范**：

| 道德要求 | 具体内容 |
|----------|----------|
| **守法遵规** | 遵守国家法律法规和行业规范 |
| **公正独立** | 独立、公正地开展监理工作 |
| **廉洁从业** | 廉洁自律，不谋取私利 |
| **诚实守信** | 诚实守信，维护行业信誉 |
| **保密义务** | 对项目信息保密 |
| **回避制度** | 与被监理单位有利害关系时回避 |

## 三、通俗例子

<Card title="监理组织例子" icon="information">
就像一个医院的组织架构：
- 院长（总监理工程师）→ 全面管理
- 各科室主任（专业监理工程师）→ 专业负责
- 主治医生、护士（监理员）→ 具体执行
</Card>

<Card title="职业道德例子" icon="information">
就像医生不能收红包一样，监理工程师也不能收受承建单位的贿赂，必须公正独立地开展监理工作。
</Card>

## 六、重点总结

### 本节必记考点 ⭐⭐⭐

1. **组织设置五原则**：专业分工、层次适当、统一指挥、跨度合理、精干高效
2. **监理人员层级**：总监理工程师→总监代表→专业监理工程师→监理员
3. **职业道德要求**：守法遵规、公正独立、廉洁从业、诚实守信、保密回避
4. **总监理工程师职责**：全面管理、人员调配、审核批准、协调沟通、签发指令、争议处理
