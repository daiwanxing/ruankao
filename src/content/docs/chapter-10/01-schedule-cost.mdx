## 10.2 网络图技术

### 一、核心讲解

#### 1. 网络图类型

| 类型 | 说明 |
|------|------|
| **AOA(Activity On Arrow)** | 箭线式网络图，活动在箭线上 |
| **AON(Activity On Node)** | 节点式网络图，活动在节点上（常用） |

#### 2. 关键路径法（CPM）

<MemoryCard 
  title="关键路径特征"
  formula="总时差=0的活动链，决定项目最短工期"
  importance={5}
  category="formula"
/>

**关键概念**：
- **最早开始时间(ES)**：活动最早可以开始的时间
- **最早结束时间(EF)**：EF = ES + 活动历时
- **最晚开始时间(LS)**：不影响工期的最晚开始时间
- **最晚结束时间(LF)**：不影响工期的最晚结束时间
- **总时差(TF)**：TF = LS - ES = LF - EF
- **自由时差(FF)**：不影响紧后活动最早开始的最大延迟

<Card title="关键路径计算步骤" icon="information">
1. **正向计算**：计算ES和EF
2. **反向计算**：计算LS和LF
3. **计算时差**：TF = LS - ES
4. **找关键路径**：TF = 0的活动链
</Card>

#### 3. 计划评审技术（PERT）

用于处理活动历时不确定的情况：

**期望历时 = (最乐观+4×最可能+最悲观) / 6**

## 10.3 进度控制

### 一、核心讲解

#### 1. 进度压缩技术

<MemoryCard 
  title="进度压缩两方法"
  formula="赶工（增加资源）、快速跟进（并行作业）"
  importance={4}
  category="key-point"
/>

| 方法 | 说明 | 风险 |
|------|------|------|
| **赶工(Crashing)** | 增加资源缩短活动历时 | 增加成本 |
| **快速跟进(Fast Tracking)** | 并行执行原本顺序的活动 | 增加返工风险 |

## 10.4 成本管理

### 一、核心讲解

#### 1. 成本管理过程

<Card title="成本管理四过程" icon="information">
1. **成本估算**：估算完成活动所需资源成本
2. **成本预算**：汇总成本估算，建立成本基准
3. **成本控制**：监控成本状态，管理成本变更
4. **成本优化**：寻求降低成本的方法
</Card>

#### 2. 成本估算方法

| 方法 | 说明 | 精度 |
|------|------|------|
| **类比估算** | 参考类似项目 | 粗略 |
| **参数估算** | 根据历史数据和参数 | 中等 |
| **自下而上估算** | 逐项估算后汇总 | 精确 |
| **三点估算** | (乐观+4×最可能+悲观)/6 | 较精确 |

## 10.5 挣值分析（EVM）

### 一、核心讲解

#### 1. 挣值三要素

<MemoryCard 
  title="挣值三要素"
  formula="PV（计划值）、EV（挣值）、AC（实际成本）"
  importance={5}
  category="formula"
/>

| 要素 | 说明 |
|------|------|
| **PV (Planned Value)** | 计划价值，计划应完成工作的预算成本 |
| **EV (Earned Value)** | 挣值，已完成工作的预算成本 |
| **AC (Actual Cost)** | 实际成本，已完成工作的实际成本 |

#### 2. 挣值分析公式

<MemoryCard 
  title="挣值分析核心公式"
  formula="CV=EV-AC（成本偏差）、SV=EV-PV（进度偏差）、CPI=EV/AC（成本绩效）、SPI=EV/PV（进度绩效）"
  importance={5}
  category="formula"
/>

| 指标 | 公式 | 含义 |
|------|------|------|
| **CV (成本偏差)** | EV - AC | 大于0：节约成本；小于0：超支 |
| **SV (进度偏差)** | EV - PV | 大于0：进度超前；小于0：进度滞后 |
| **CPI (成本绩效指数)** | EV / AC | 大于1：成本节约；小于1：成本超支 |
| **SPI (进度绩效指数)** | EV / PV | 大于1：进度超前；小于1：进度滞后 |

#### 3. 完工预测

| 公式 | 说明 |
|------|------|
| **ETC** | 完工尚需成本 |
| **EAC** | 完工估算 = AC + ETC |
| **VAC** | 完工偏差 = BAC - EAC |

**常用EAC计算**：
- 按当前CPI：EAC = BAC / CPI
- 按原计划：EAC = AC + (BAC - EV)

## 重点总结

### 本章必记考点 ⭐⭐⭐⭐⭐

1. **进度工具**：甘特图、里程碑图、网络图
2. **关键路径**：总时差=0的活动链，决定最短工期
3. **时差公式**：TF = LS - ES = LF - EF
4. **PERT期望历时**：(乐观+4×最可能+悲观)/6
5. **进度压缩**：赶工(增资源)、快速跟进(并行)
6. **挣值三要素**：PV、EV、AC
7. **挣值公式**：CV=EV-AC、SV=EV-PV、CPI=EV/AC、SPI=EV/PV
8. **判断标准**：CPI/SPI大于1为好，小于1为差；CV/SV大于0为好，小于0为差
