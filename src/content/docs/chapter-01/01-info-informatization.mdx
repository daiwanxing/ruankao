## 一、教学目标

掌握信息的基本概念、信息化的内涵与特征，以及国家信息化战略规划，为后续学习打下坚实基础。

## 二、核心讲解

### 1. 信息的概念

**数据与信息的关系**：
- **数据**：由原始事实组成，是信息的载体
- **DIKW模型**：数据(Data) → 信息(Information) → 知识(Knowledge) → 智慧(Wisdom)

**有价值的信息特征（12个特征）**：
便捷性、准确性、完整性、经济性、灵活性、相关性、可靠性、安全性、简单性、及时性、可检验性

### 2. 信息化的基本内涵

<MemoryCard
  title="信息化六维度口诀"
  formula="主权、常识、空域、手工、涂改、飙升"
  importance={5}
  category="formula"
/>

| 维度 | 内涵 |
|------|------|
| **主体** | 全体社会成员（政府、企业、事业、团体和个人） |
| **时域** | 是一个长期的过程 |
| **空域** | 政治、经济、文化、军事和社会的一切领域 |
| **手段** | 基于现代信息技术的先进社会生产工具 |
| **途径** | 创建信息时代的社会生产力 |
| **目标** | 使国家的综合实力、社会的文明素质和人民的生活质量全面提升 |

### 3. 国家信息化战略与规划

**组织信息化趋势**：产品信息化、产业信息化、社会生活信息化、国民经济信息化

**三步走战略**：
- **第一步（到2020年）**：核心关键技术部分领域达到国际先进水平
- **第二步（到2025年）**：建成国际领先的移动通信网络
- **第三步（到21世纪中叶）**：信息化全面支撑社会主义现代化建设

## 三、通俗例子

<Card title="DIKW模型例子" icon="information">
- **数据**：你今天走了10,000步（原始数字）
- **信息**：今天比昨天多走了2,000步（有了比较和意义）
- **知识**：连续一周运动步数增加，说明身体状况在改善（规律认知）
- **智慧**：知道如何通过合理运动来保持健康（运用知识的能力）
</Card>

<Card title="信息化主体例子" icon="information">
就像一个城市的发展不仅需要政府的规划，还需要企业、学校、医院、社区等各方参与，信息化的主体也是全社会成员。
</Card>
