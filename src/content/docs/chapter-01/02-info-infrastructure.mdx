## 一、教学目标

掌握新型基础设施建设（新基建）七大领域、云计算服务类型、大数据4V特征、物联网特点、工业互联网四大体系、区块链核心技术、人工智能分类以及VR/AR区别。

## 二、核心讲解

### 1. 新型基础设施建设（新基建）

**定义**：以新发展理念为引领，以技术创新为驱动，以信息网络为基础，面向高质量发展，提供数字转型、智能升级、融合创新服务的基础设施体系。

**新基建七大领域**：5G基建、特高压、城际高速铁路和城际轨道交通、新能源汽车充电桩、大数据中心、人工智能、工业互联网

**三大分类**：

| 类型 | 关键词 | 主要内容 |
|------|--------|----------|
| 新型基础设施 | 技术新 | 5G、物联网、工业互联网、卫星互联网、人工智能、云计算、区块链、数据中心、智能计算中心 |
| 融合基础设施 | 应用新 | 智能交通基础设施、智慧能源基础设施 |
| 创新基础设施 | 平台新 | 重大科技基础设施、科教基础设施、产业技术创新基础设施 |

### 2. 云计算

**特征**：超大规模、高可扩展性、虚拟化、高可靠性、通用性、廉价性、灵活定制

**服务类型分类**：

| 层次 | 全称 | 说明 | 比喻 |
|------|------|------|------|
| **IaaS** | 基础设施即服务 | 提供服务器、存储和网络硬件 | 租房（提供基本居住空间） |
| **PaaS** | 平台即服务 | 提供数据库、中间件及开发工具 | 租房+装修（提供装修好的房子） |
| **SaaS** | 软件即服务 | 将应用程序以服务形式提供给用户 | 住酒店（直接入住享受服务） |

**部署范围**：公有云、私有云、混合云

### 3. 大数据

**4V特征**：

| 特征 | 英文 | 含义 |
|------|------|------|
| **规模大** | Volume | 数据量巨大，通常在PB级别以上 |
| **多样性** | Variety | 数据类型多样，包括结构化、半结构化、非结构化 |
| **速度快** | Velocity | 数据产生和处理速度快 |
| **价值密度低** | Value | 单条数据价值密度低，但总体价值高 |

**结构分类**：
- **结构化数据**：符合关系型数据库，二维表
- **半结构化数据**：不符合关系型数据库（如XML、JSON）
- **非结构化数据**：没有固定结构（如文本、图片、视频）

**Hadoop生态系统**：
- **HDFS**：分布式文件存储系统
- **MapReduce**：计算模型，用于大规模数据集并行运算
- **Yarn**：资源管理

### 4. 物联网（IoT）

**特点**：互连、识别与通信、智能化

**产业六个特点**：感知识别普适化、异构设备互连化、互联网终端规模化、管理调控智能化、应用服务链条化、经济发展跨越化

**分类**：
- **按部署方式**：私有物联网、公有物联网、社区物联网、混合物联网
- **按传输速率**：高速率、中速率、低速率业务

### 5. 工业互联网

**四大体系**：

| 体系 | 作用 |
|------|------|
| **网络体系（基础）** | 网络互连、数据互通、标识解析 |
| **平台体系（中枢）** | 边缘层、IaaS层、PaaS层、SaaS层 |
| **数据体系（要素）** | 价值创造的源泉 |
| **安全体系（保障）** | 设备、控制、网络、平台、工业应用的整体保护 |

### 6. 区块链

**特点**：多方协作、不可篡改、可追溯

**分类**：公有链、私有链、联盟链

**核心技术**：

| 技术 | 说明 |
|------|------|
| 分布式存储 | 分布式公共账本，点对点记账系统 |
| 共识机制 | PoW、PoS、DPoS、Paxos、PBFT等 |
| 智能合约 | 基于预定义事件触发、不可篡改、自动执行的计算机协议 |
| 加密算法 | 对称加密（如DES）和非对称加密（如RSA） |
| 跨链技术 | 将区块链数据安全转移到另一条区块链 |
| 分片技术 | 将数据库分割为不同片区以达到系统扩容 |

### 7. 人工智能

**分类**：
- **弱人工智能**：不能真正实现推理和解决问题，无自主意识（如语音识别、图像识别）
- **强人工智能**：真正能思维的智能机器，有自我意识（目前尚未实现）

**特征**：由人类设计，为人类服务；能感知环境；有适应特性

### 8. 虚拟现实（VR）和增强现实（AR）

**区别**：

| 区别 | VR | AR |
|------|----|----|
| **设备** | 位置追踪器、数据手套、动作捕捉系统、数据头盔 | 3D摄像头，只要安装AR软件，带摄像头的产品都可以进行AR体验 |
| **技术** | 绘图相关技术，关注沉浸感 | 计算机视觉技术，对真实场景进行3D建模再处理，重视CPU处理能力 |
| **应用场景** | 游戏、娱乐、教育、社交 | 生活、工作、生产等场景 |

## 三、通俗例子

<Card title="云计算例子" icon="information">
- **IaaS**：像租房子，提供基本的居住空间
- **PaaS**：像租房+装修，提供装修好的房子
- **SaaS**：像住酒店，直接入住享受服务
</Card>

<Card title="大数据例子" icon="information">
就像淘宝分析你的购物行为，虽然单看一次购买行为数据量很小，但通过分析海量用户的购买数据，就能发现消费趋势，这就是大数据"4V"的体现。
</Card>

<Card title="物联网例子" icon="information">
智能家居就是物联网的典型应用，冰箱、空调、灯光等设备互相连接，可以通过手机远程控制。
</Card>

<Card title="区块链例子" icon="information">
就像村里记账，以前只有村长一个人记账（中心化），现在每家每户都有一本一模一样的账本（分布式），如果有人想篡改，需要同时修改大部分账本，这几乎不可能。
</Card>

## 六、重点总结

### 本节必记考点 ⭐⭐⭐⭐⭐

1. **新基建七大领域**：5G、特高压、城轨、充电桩、大数据、AI、工业互联
2. **云计算三层**：IaaS(基础设施)、PaaS(平台)、SaaS(软件)
3. **大数据4V**：Volume(规模大)、Variety(多样性)、Value(价值密度低)、Velocity(速度快)
4. **物联网三特点**：互连、识别与通信、智能化
5. **工业互联网四大体系**：网络(基础)、平台(中枢)、数据(要素)、安全(保障)
6. **区块链三大特点**：多方协作、不可篡改、可追溯
7. **AI分类**：弱AI(无自主意识)、强AI(有自主意识)
