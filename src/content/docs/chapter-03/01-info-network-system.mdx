## 3.1 信息网络系统体系框架和OSI七层模型

### 一、教学目标

掌握信息网络系统体系框架的五大部分和OSI七层模型各层的功能及特点。

### 二、核心讲解

#### 1. 信息网络系统体系框架模型

| 组成部分 | 内容 |
|----------|------|
| **网络传输平台** | 传输、路由、交换、有线和无线接入等设备和系统 |
| **网络和应用服务平台** | DNS、地址分配系统、OA、WWW、电子邮件、视频会议等 |
| **安全服务平台** | 信息加解密、防火墙、入侵检测、漏洞扫描、病毒查杀、安全审计、数字证书等 |
| **网络管理和维护平台** | 整个信息网络系统的管理和维护，运营系统 |
| **环境系统** | 机房建设、环境监控、智能安防、节能降耗、综合布线等 |

#### 2. OSI七层模型

<MemoryCard 
  title="OSI七层模型记忆口诀"
  formula="物链网传会表应（物理、数据链路、网络、传输、会话、表示、应用）"
  importance={5}
  category="formula"
/>

| 层级 | 主要功能 | 关键设备/协议 |
|------|----------|---------------|
| **应用层** | 直接向用户提供服务 | HTTP、FTP、SMTP |
| **表示层** | 数据编解码、加密解密、格式转换 | 数据格式化 |
| **会话层** | 建立、管理、拆除会话连接 | 会话管理 |
| **传输层** | 端到端可靠传输、差错控制、流量控制 | TCP、UDP |
| **网络层** | 路由选择、逻辑寻址 | IP、路由器 |
| **数据链路层** | 数据包格式、物理寻址、差错控制 | 交换机、网卡 |
| **物理层** | 定义物理设备间的数据传输方法 | 集线器、双绞线 |

### 三、通俗例子

<Card title="OSI七层模型寄快递例子" icon="information">
- **应用层**：你要寄的东西（文件、包裹）
- **表示层**：把东西打包、加密
- **会话层**：确认收货、签收过程
- **传输层**：确保快递安全送达
- **网络层**：选择快递路线（顺丰、圆通）
- **数据链路层**：快递包装、地址标签格式
- **物理层**：快递员、交通工具、道路
</Card>

## 3.2 TCP/IP协议

### 一、核心讲解

#### 1. TCP/IP四层模型

| TCP/IP层 | 对应OSI层 | 主要协议 |
|----------|-----------|----------|
| **应用层** | 应用层+表示层+会话层 | HTTP、FTP、SMTP、DNS |
| **传输层** | 传输层 | TCP、UDP |
| **网际层** | 网络层 | IP、ICMP、ARP |
| **网络接口层** | 数据链路层+物理层 | 以太网、WiFi |

#### 2. IP地址

<MemoryCard 
  title="IPv4地址分类"
  formula="A类(0)、B类(10)、C类(110)、D类(1110)、E类(1111)"
  importance={4}
  category="key-point"
/>

| 类别 | 首字节范围 | 网络位/主机位 | 用途 |
|------|-----------|---------------|------|
| A类 | 1-126 | 8/24 | 大型网络 |
| B类 | 128-191 | 16/16 | 中型网络 |
| C类 | 192-223 | 24/8 | 小型网络 |
| D类 | 224-239 | - | 组播 |
| E类 | 240-255 | - | 保留 |

#### 3. TCP与UDP对比

| 特性 | TCP | UDP |
|------|-----|-----|
| **连接** | 面向连接 | 无连接 |
| **可靠性** | 可靠传输 | 不可靠传输 |
| **速度** | 较慢 | 较快 |
| **应用场景** | 文件传输、邮件 | 视频、语音 |

## 3.3 网络设备

### 一、核心讲解

| 设备 | 工作层次 | 功能 |
|------|----------|------|
| **集线器** | 物理层 | 信号放大和转发 |
| **交换机** | 数据链路层 | MAC地址转发 |
| **路由器** | 网络层 | IP地址路由选择 |
| **网关** | 传输层及以上 | 协议转换 |

## 重点总结

### 本章必记考点 ⭐⭐⭐⭐⭐

1. **网络框架五平台**：传输、应用服务、安全、管理维护、环境
2. **OSI七层**：物链网传会表应（从下到上）
3. **TCP/IP四层**：网络接口层、网际层、传输层、应用层
4. **IPv4分类**：A类(1-126)、B类(128-191)、C类(192-223)
5. **TCP vs UDP**：TCP面向连接可靠，UDP无连接快速
6. **网络设备**：集线器(物理层)、交换机(链路层)、路由器(网络层)
