## 一、教学目标

掌握机房建设、综合布线系统、机房监控系统和节能降耗等内容。

## 二、核心讲解

### 1. 机房建设

**机房组成**：
- 机房基础环境
- 供配电系统
- 安防系统
- 消防系统
- 环境监控系统
- 精密空调系统

### 2. 综合布线系统

**六个子系统**：

| 子系统 | 说明 |
|--------|------|
| **工作区子系统** | 终端设备连接到信息插座的部分 |
| **配线子系统** | 楼层配线架到工作区信息插座的部分 |
| **干线子系统** | 楼层间垂直连接的部分 |
| **管理子系统** | 对布线系统进行管理的部分 |
| **设备间子系统** | 集中安装设备的房间 |
| **建筑群子系统** | 建筑物之间连接的部分 |

**记忆口诀**：工配干管设建

### 3. 机房监控系统

**监控内容**：
- 温度湿度监控
- 门禁监控
- 设备监控
- 电力监控
- 消防监控

### 4. 节能降耗

**三个方面**：
- **选址**：选择适宜的地理位置
- **布局**：合理的机房布局设计
- **设备**：采用节能设备

## 三、通俗例子

<Card title="综合布线例子" icon="information">
就像装修房子：
- **工作区**：各个房间的电源插座和网络接口
- **配线**：房间内的电线和网线
- **干线**：楼层之间的主线路
- **管理**：配电箱和弱电箱
- **设备间**：小区的机房
- **建筑群**：小区之间的连接道路
</Card>
