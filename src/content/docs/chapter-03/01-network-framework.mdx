## 一、教学目标

掌握信息网络系统体系框架的五大部分和OSI七层模型各层的功能及特点。

## 二、核心讲解

### 1. 信息网络系统体系框架模型

| 组成部分 | 内容 |
|----------|------|
| **网络传输平台** | 传输、路由、交换、有线和无线接入等设备和系统 |
| **网络和应用服务平台** | DNS、地址分配系统、OA、WWW、电子邮件、视频会议等 |
| **安全服务平台** | 信息加解密、防火墙、入侵检测、漏洞扫描、病毒查杀、安全审计、数字证书等 |
| **网络管理和维护平台** | 整个信息网络系统的管理和维护，运营系统 |
| **环境系统** | 机房建设、环境监控、智能安防、节能降耗、综合布线等 |

### 2. OSI七层模型

<MemoryCard
  title="OSI七层模型记忆口诀"
  formula="物链网传会表应（物理、数据链路、网络、传输、会话、表示、应用）"
  importance={5}
  category="formula"
/>

| 层级 | 主要功能 | 关键设备/协议 |
|------|----------|---------------|
| **应用层** | 直接向用户提供服务 | HTTP、FTP、SMTP |
| **表示层** | 数据编解码、加密解密、格式转换 | 数据格式化 |
| **会话层** | 建立、管理、拆除会话连接 | 会话管理 |
| **传输层** | 端到端可靠传输、差错控制、流量控制 | TCP、UDP |
| **网络层** | 路由选择、逻辑寻址 | IP、路由器 |
| **数据链路层** | 数据包格式、物理寻址、差错控制 | 交换机、网卡 |
| **物理层** | 定义物理设备间的数据传输方法 | 集线器、双绞线 |

## 三、通俗例子

<Card title="OSI七层模型寄快递例子" icon="information">
- **应用层**：你要寄的东西（文件、包裹）
- **表示层**：把东西打包、加密
- **会话层**：确认收货、签收过程
- **传输层**：确保快递安全送达
- **网络层**：选择快递路线（顺丰、圆通）
- **数据链路层**：快递包装、地址标签格式
- **物理层**：快递员、交通工具、道路
</Card>

<Card title="信息网络系统框架例子" icon="information">
就像一个城市的基础设施：
- **网络传输平台**：道路、交通工具
- **网络和应用服务平台**：邮局、商店、银行等服务场所
- **安全服务平台**：警察局、保安公司、监控系统
- **网络管理和维护平台**：交通管理中心、市政管理部门

</Card>

## 六、重点总结

### 本节必记考点 ⭐⭐⭐⭐⭐

1. **网络框架五平台**：网络传输、网络应用服务、安全服务、网络管理和维护、环境系统
2. **OSI七层模型**：物理层→数据链路层→网络层→传输层→会话层→表示层→应用层
3. **各层功能**：物理层（硬件基础）、数据链路层（数据包格式）、网络层（路由选择）、传输层（端到端连接）、会话层（会话管理）、表示层（数据转换）、应用层（用户服务）
